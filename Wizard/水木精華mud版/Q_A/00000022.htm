︰   1. uncompress mudbin.tgz (such as in d:\mud) and 
︰       es2win95.tgz( ie. directary is d:\lpmud) 
use winzip etc... 
︰   2. copy d:\lpmud\adm\etc\config.ES2  to d:\mud 
︰   3 modify config.ES2 about file location: 
︰     mudlib directory : d:\lpmud 
︰     binary directory : d:\mud 
add mudos.ini to your windows dir,such as c:\windows 
[init] 
path=d:\mud 
param=config.ES2 
;that means the param is d:\mud\config.es2 
 
︰   4. run mudos.ese config.ES2  
run mudos.exe,no param. 
︰ You can use other mud lib .maybe need some modify. 
See debug.log(/log/debug.log) or mud.dat(int the mudos dir) 
for details.
