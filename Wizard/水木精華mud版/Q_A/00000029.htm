任何物件在create的時候都由master object 
的一個creator_file之類的函數給出uid，而且 
沒有setuid函數，就是原則上不能修改。export_uid就是 
把this_object的euid export到export_uid的參數上作為 
uid。

        名稱: 
                export_uid() -  設定一個物件的使用者識別名稱 (user id, 
                                uid). 
        用法: 
                int export_uid( object ob ); 
 
                整數 export_uid( 物件 ob ); 
        用法: 
                設定 ob 的使用者識別名稱 (uid) 為 this_object() 的 
                有效使用者識別名稱 (effective user id, effective uid). 
                這個函式只有在 ob 的有效使用者識別名稱為 0 時有效. 
        參考: 
                this_object(3), seteuid(3), getuid(3), geteuid(3), 
                previous_object(3), valid_seteuid(4)

                Spock @ FF      97.Feb.12.              (printed 3/16/95)
