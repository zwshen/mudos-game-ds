附圖：主地形－－氣候（部分） 
 
 
    熱帶雨林氣候：全年高溫悶熱，多雨潮濕，風力微弱，幹季不明顯。最涼月平均氣溫 
攝氏18度以上，年平均氣溫24∼30度，年平均降水量1500∼3000毫米以上。 
 
    全年高溫悶熱，多雨潮濕，風力微弱 
 
    熱帶季風氣候 
 
    亞熱帶季風性濕潤氣候, 受季風影響，冬季溫和少雨，夏熱多雨 
 
    熱帶草原氣候：暖季多雨，涼季幹燥。最熱月平均氣溫攝氏25∼30度以上，最涼月平 
均氣溫攝氏15∼18度，年平均降水量200 ∼1000多毫米，幹濕季節分明。 
 
    暖季多雨，涼季幹燥 
 
    熱帶沙漠氣候：高溫少雨，最熱月平均氣溫攝氏32∼36度，最涼月平均氣溫攝氏10度 
以上。年平均降水量150 毫米以下，有的地區甚至連續數年無雨。 
 
    高溫少雨 
 
    亞熱帶季風性濕潤氣候 
 
    冬季溫和少雨，夏熱多雨 
 
    地中海氣候：冬季溫和多雨，夏熱幹燥。最冷月平均氣溫攝氏零度以上，最熱月平均 
氣溫攝氏22∼28度。年平均降水量 300∼1000毫米。 
 
    冬季溫和多雨，夏熱幹燥 
 
    ＊溫帶海洋性氣候： 
 
    溫帶季風氣候：夏季盛行風從海洋吹向大陸，暖熱多雨；冬季盛行風從內陸吹向海洋， 
寒冷幹燥。最冷月平均氣溫可低達攝氏零下二十度，最熱月平均氣溫攝氏二十至二十五度， 
年平均降水量五百至一千毫米。 
 
    溫帶大陸性氣候 
 
    ＊極地氣候：冬長而冷，夏短而涼。最熱月平均氣溫不超過攝氏十至十二度，但高於 
零度。 
 
    高山氣候：因高度和地形影響而形成的山區特殊氣候。由地理位置、海拔高度、坡谷 
方位、山峰分布及地域性氣候條件等因素所決定。山地垂直起伏達數千米時，氣候垂直差 
異特別明顯，氣溫及其日較差、年較差均隨高度遞減。在對流圈內每升高一百米氣溫平均 
降低攝氏0.6 度。在不同緯度地帶的山地及其向、背風坡的各高度上，具有不同的氣候種 
類。 
 
    我國氣候特徵： 
 
我國大部分地區位於北溫帶和亞熱帶，屬季風氣候區。由於領土遼闊、地形複雜，各地距 
海遠近差異較大，形成了氣候複雜多樣的特點，如黑龍江北部全年無夏，海南島無冬，淮 
河流域四季分明，雲貴高原南部四季如春，青藏高原西部終年積雪，西北內陸夏熱冬冷， 
日較差大。總的特點是：全國夏季普遍高溫，降水較多；冬季南北溫差大。年平均降水量 
從東南的一千五百毫米以上，向西北遞減到五十毫米。 
 
 
                                    天氣現像 
 
    雲由飄浮在空中的水蒸氣遇冷變成細小水滴而形成。水汽形成是有條件的，空氣必須 
作上升運動，空氣降溫使溫度達到飽和，還必須有充足的水汽來源，才能維持雲的存在和 
發展，常見雲的現像有少雲、多雲、陰天。 
 
    霧是在氣溫下降時空氣中所含水蒸氣凝固成小水點，浮在接近地面空氣中而形成。霧 
一般發生在晚秋和冬天的早晨。在晴朗的晚間，地面向外輻射熱量，溫度降低，靠近地面 
的空氣溫度隨之降低，空氣溫度越低，可容納的水汽就越少，於是水汽結或凝集成小水滴 
成小冰晶，由於有微風，便懸浮在空中，將一切景物籠罩在朦矓之中。 
 
    雨是從雲中降向地面的水滴。水蒸氣從池塘、河流、海洋、樹林、農田等各種地方蒸 
發到天空，並在高空遇冷變成小水滴團，這便是平時看到的雲。當雲層變厚，水滴也變大， 
不能繼續飄浮在天空而下落便成為雨。常見雨的現像有小雨、大雨、暴雨。 
 
    下雨時會伴隨有閃電和雷聲。閃電是雲與雲之間或者雲與地面之間所發生的放電現像。 
閃電的發生，是因為雲中每個小水滴表面都帶有正電荷或負電荷，通常正電荷集中在雲的 
上部，負電荷聚集在雲的底部，大地因為受雲底部電荷吸引，很多正電荷都跑到地表上來， 
正電荷和負電荷相互吸引，當正負電荷分別增加到一定數量時，吸引的力量也變生相當大， 
電荷便穿透空氣，互相結合而產生放電現像。因此，狂風暴雨尤其是雷電時，應待在室內， 
關好門窗；如在野外，不要靠近高大的物體；在曠地時應蹲在地上，僅用雙腳觸地，避免 
接觸金屬物體；人群要及時散開；游泳和划船者要盡快離開水面。打雷是雲層放電時發出 
的響聲。當閃電發生時，會發出強烈的光，同時產生高溫，使閃電附近的空氣因劇烈受熱 
而突然膨脹，雲中水滴因為高熱也會突然氣化膨脹，於是發出巨大的爆炸般的響聲。 
 
    雨後會出現彩虹，彩虹是大氣中一種光的現像。天空中的小水珠經日光照射發生折射 
和反射作用而形成的弧形彩帶，我們便稱之為彩虹，彩虹由外向內分別呈現出紅、橙、黃、 
綠、青、藍、紫七種色彩光。 
 
    霜是在氣溫降到攝氏零度以下時，靠近地面空氣中所含的水蒸氣在地面物體上凝結成 
的白色冰晶。霜不是從天上降下來的。白天，大地受到太陽的照射，溫度比較高，土壤中 
的水分不斷向外蒸發，接近地面的空氣便有了一定的水汽，晚上，溫度驟然下降，這些水 
汽接觸到寒冷的物體表面，結成冰晶，這就是霜，尤其是在疏鬆的土壤表面，結成冰霜後， 
還能將地下的水分吸到表面上來，將地面上的冰向上推起，形成很多霜柱。 
 
    雪是在氣溫降到攝氏零度以下時，空氣中的水蒸氣凝結而成，從天空降落的白色結晶。 
雪通常是六角形的。水蒸氣在凝結時，溫度很低的雲開始由極小的冰晶和過冷卻水滴組成， 
水滴不斷蒸發成水汽，在冰晶上凝集，使冰晶長大形成雪花。雪花自身重了，就從空中落 
下來，就形成了雪。常見雪的現像有小雪、大雪、暴雪。 
 
    風是與地面大致平行的空氣流動，是由於氣壓分布不均勻而產生的。風的產生，是由 
於太陽輻射地面各部分的能量不相等，而造成地球各個部分冷熱不均勻，氣壓大小不一致， 
空氣總是從氣壓高的地方向氣壓低的地方流動，便形成我們感覺到的風。風有風向，例如 
東風、南風、西風、北風、東北風、東南風、西南風、西北風。風有速度並由此區分級別： 
０級，無風；１級，軟風；２級，輕風；３級，微風；４級，和風；５級，勁風；６級， 
強風；７級，疾風；８級，大風；９級，烈風；１０級，狂風；１１級，暴風；１２級颶 
風。 
 
    霞是日出日落時出現的天氣現像。那時太陽光要穿過的空氣層才能照射到地平線上空， 
太陽光在大氣中走的路程很長，陽光中的紫色和藍色光被散射得很厲害，到達地平線時已 
所剩不多，陽光中剩下的只有紅光，綠光和黃光，這些光線經地平線上空的空氣分子、水 
汽、塵埃的散射後，我們便看到瑰麗的彩霞。晚霞出現通常表明第二天天氣睛朗。這是由 
於霞的鮮艷程度與大氣中水汽含量有關，因此霞對天氣變化具有預示意義。

// 函數：顯示天氣狀態 
void show_nature_phase( string sPhase ) 
{ 
        string sRetMsg; 
        int i; 
 
        // 生成天氣描述字串 
        switch( mpCliPhase[sPhase]["climate"] ) 
        { 
                case "晴"   :  
                case "少雲" : sRetMsg = "陽光直直照射下來。";   break; 
                case "多雲" : 
                case "陰"   : sRetMsg = "陽光被雲層遮住了。";   break; 
                case "霧"   : sRetMsg = "霧氣向四周瀰漫開來。"; break; 
                case "霜"   : sRetMsg = "地上到處開始結起霜。"; break; 
                case "霞"   : sRetMsg = "霞光映紅了天。";       break; 
                case "小雪" : 
                case "雪"   : sRetMsg = "天上開始飄起雪花。" ;  break; 
                case "大雪" : 
                case "暴雪" : sRetMsg = "天上開始下起大雪。" ;  break; 
                case "小雨" : 
                case "雨"   : sRetMsg = "天上開始下起小雨。" ;  break; 
                case "大雨" : 
                case "暴雨" : sRetMsg = "天上開始下起大雨。" ;  break; 
        } 
 
        // 生成溫度描述字串 
        i = mpCliPhase[sPhase]["temperature"]; 
        if( i < 0 ) sRetMsg += "天氣變得非常冷。\n"; 
        else if( i < 4 ) sRetMsg += "天氣變得很冷。\n"; 
        else if( i < 10 ) sRetMsg += "天氣變得有些冷。\n"; 
        else if( i < 15 ) sRetMsg += "天氣變得有些涼。\n"; 
        else if( i < 20 ) sRetMsg += "天氣變得有些暖。\n"; 
        else if( i < 25 ) sRetMsg += "天氣變得有些熱。\n"; 
        else if( i < 30 ) sRetMsg += "天氣變得很熱。\n"; 
        else sRetMsg += "天氣變得非常熱。\n"; 
 
        // 生成風力描述字串 
        switch( mpCliPhase[sPhase]["wind"] ) 
        { 
                case 0 : sRetMsg += "身邊的風完全停住了。" ; break; 
                case 1 : sRetMsg += "你身邊吹起軟軟的風。" ; break; 
                case 2 : sRetMsg += "你身邊吹起輕輕的風。" ; break; 
                case 3 : sRetMsg += "你身邊吹起微微的風。" ; break; 
                case 4 : sRetMsg += "你身邊吹起習習和風。" ; break; 
                case 5 : sRetMsg += "你身邊呼呼吹起勁風。" ; break; 
                case 6 : sRetMsg += "你四周呼呼刮起強風。" ; break; 
                case 7 : sRetMsg += "你四周嗚嗚刮起疾風。" ; break; 
                case 8 : sRetMsg += "你四周嗚嗚刮起大風。" ; break; 
                case 9 : sRetMsg += "你四周嗚嗚刮起烈風。" ; break; 
                case 10: sRetMsg += "你四周嗚嗚刮起狂風。" ; break; 
                case 11: sRetMsg += "你四周嗚嗚刮起暴風。" ; break; 
                case 12: sRetMsg += "你四周嗚嗚刮起颶風。" ; break; 
        } 
 
        // 生成雲層描述字串 
        switch( mpCliPhase[sPhase]["cloud"] ) 
        { 
                case 0 :  case 1 :  case 2 :  case 3 :  
                        sRetMsg += "天空中萬里無雲。\n"; break; 
                case 4 :  case 5 :  case 6 :  
                        sRetMsg += "幾朵淡淡的雲彩妝點著天空。\n"; break; 
                case 7 :  case 8 :  case 9 :   
                        sRetMsg += "白雲在天空中飄來飄去。\n"; break; 
                case 10 :   
                        sRetMsg += "厚厚的雲層堆積在天邊。\n"; break; 
        } 
 
        // 顯示天氣狀態 
}

        } 
 
        // 生成溫度描述字串 
        i = mpCliPhase[sPhase]["temperature"]; 
        if( i < 0 ) sRetMsg += "你覺得非常冷。\n"; 
        else if( i < 4 ) sRetMsg += "你覺得很冷。\n"; 
        else if( i < 10 ) sRetMsg += "你覺得有些冷。\n"; 
        else if( i < 15 ) sRetMsg += "你覺得有些涼。\n"; 
        else if( i < 20 ) sRetMsg += "你覺得有些暖。\n"; 
        else if( i < 25 ) sRetMsg += "你覺得有些熱。\n"; 
        else if( i < 30 ) sRetMsg += "你覺得很熱。\n"; 
        else sRetMsg += "你覺得非常熱。\n"; 
 
        // 生成風力描述字串 
        switch( mpCliPhase[sPhase]["wind"] ) 
        { 
                case 0 : sRetMsg += "一點風都沒有。" ; break; 
                case 1 : sRetMsg += "軟軟的風吹過。" ; break; 
                case 2 : sRetMsg += "輕輕的風吹過。" ; break; 
                case 3 : sRetMsg += "微微的風吹過。" ; break; 
                case 4 : sRetMsg += "習習和風吹過。" ; break; 
                case 5 : sRetMsg += "勁風呼呼吹過。" ; break; 
                case 6 : sRetMsg += "強風呼呼刮過。" ; break; 
                case 7 : sRetMsg += "疾風嗚嗚刮過。" ; break; 
                case 8 : sRetMsg += "大風嗚嗚刮過。" ; break; 
                case 9 : sRetMsg += "烈風嗚嗚刮過。" ; break; 
                case 10: sRetMsg += "狂風橫掃大地。" ; break; 
                case 11: sRetMsg += "暴風狂掃大地。" ; break; 
                case 12: sRetMsg += "颶風狂掃大地。" ; break; 
        } 
 
        // 生成雲層描述字串 
        switch( mpCliPhase[sPhase]["cloud"] ) 
        { 
                case 0 :  case 1 :  case 2 :  case 3 :  
                        sRetMsg += "天空中萬里無雲。\n"; break; 
                case 4 :  case 5 :  case 6 :  
                        sRetMsg += "幾朵淡淡的雲彩妝點著天空。\n"; break; 
                case 7 :  case 8 :  case 9 :   
                        sRetMsg += "白雲在天空中飄來飄去。\n"; break; 
                case 10 :   
                        sRetMsg += "厚厚的雲層堆積在天邊。\n"; break; 
        } 
 
        // 顯示天氣狀態 
        return sRetMsg; 
} 
 
// 函數：更新天氣狀態 
void update_nature_phase() 
{ 
        string *sList, sCliPhase, sPhase; 
        int iNewTime, iNewT, iNewWater, iNewWind, iNewCould; 
        int i, iSize, iMinT, iMaxT, iWaterR, iWindR; 
        int iSeason, iMonth, iHour; 
 
        // 取消重複調用 
        remove_call_out("update_nature_phase"); 
 
        // 獲取遊戲季節、月份、小時 
        iHour = get_game_hour(); 
        iMonth = get_game_month(); 
        switch(iMonth) 
        { 
                case 12:  case  1:  case  2:    iSeason = -1; break; 
                case  6:  case  7:  case  8:    iSeason =  1; break; 
                default:                        iSeason =  0; break; 
        } 
         
        // 對狀態終結的天氣進行轉轉 
        sList = keys(mpCliPhase); 
        for( i = 0, iSize = sizeof(sList); i < iSize ; i ++ ) 
                if( -- mpCliPhase[ sList[i] ]["time"] <= 0 ) 
                { 
                        // 獲取遊戲季節氣候參數 
                        sCliPhase = sList[i]; 
                        switch(iSeason) 
                        { 
                                case -1 : 
                                        iMinT = mpCliPhase[sCliPhase]["1_TEMP_RANGE"][0]; 
                                        iMaxT = mpCliPhase[sCliPhase]["1_TEMP_RANGE"][1]; 
                                        iWaterR = mpCliPhase[sCliPhase]["1_WATER_RATE"]; 
                                        iWindR = mpCliPhase[sCliPhase]["1_WIND_RATE"]; 
                                        break; 
                                case  1 : 
                                        iMinT = mpCliPhase[sCliPhase]["7_TEMP_RANGE"][0]; 
                                        iMaxT = mpCliPhase[sCliPhase]["7_TEMP_RANGE"][1]; 
                                        iWaterR = mpCliPhase[sCliPhase]["7_WATER_RATE"]; 
                                        iWindR = mpCliPhase[sCliPhase]["7_WIND_RATE"]; 
                                        break; 
                                default : 
                                        iMinT = mpCliPhase[sCliPhase]["X_TEMP_RANGE"][0]; 
                                        iMaxT = mpCliPhase[sCliPhase]["X_TEMP_RANGE"][1]; 
                                        iWaterR = mpCliPhase[sCliPhase]["X_WATER_RATE"]; 
                                        iWindR = mpCliPhase[sCliPhase]["X_WIND_RATE"]; 
                                        break; 
                        } 
 
                        // 隨機生成連續變化氣溫 
                        iNewT = mpCliPhase[sCliPhase]["temperature"]; 
                        iNewT += random(5) - 2; 
                        if( iNewT < iMinT )  iNewT = iMinT; 
                        if( iNewT > iMaxT )  iNewT = iMaxT; 
                        // 隨機生成當前風雨參數 
                        iNewWind = random(iWindR); 
                        if( random(20) > iWaterR ) iNewWater = 0; 
                        else iNewWater = random(iWaterR); 
                        // 隨機生成連續變化雲層 
                        iNewCould += random(5) - 2; 
                        if( iNewCould < 0 ) iNewCould = 0; 
                        if( iNewCould > 10 ) iNewCould = 10; 
                        // 隨機生成天氣持繼時間 
                        iNewTime = random(10) + 1; 
                         
                        // 根據降水狀態進行處理 
                        sPhase = "晴"; 
                        if(iNewWater)  
                        { 
                                // 降水天氣通常天空有雲 
                                iNewCould = iNewWater * 2 + 2; 
                                if( iNewCould > 10 )  iNewCould = 10; 
                                // 根據季節校正降水溫度 
                                if( iSeason == -1 && iMinT < 0 && iNewT >= 0 )  
                                        iNewT = ( iNewT + iMinT ) / 2; 
                                // 根據溫度判斷下雨下雪 
                                if( iNewT > 0 )  
                                        switch(iNewWater) 
                                        { 
                                                case 1 :  case 2 :  case 3 :  
                                                        sPhase = "小雨"; break; 
                                                case 4 :  case 5 :  case 6 :  
                                                        sPhase = "雨";   break; 
                                                case 7 :  case 8 :  case 9 :   
                                                        sPhase = "大雨"; break; 
                                                case 10 :   
                                                        sPhase = "暴雨"; break; 
                                        } 
                                else 
                                        switch(iNewWater) 
                                        { 
                                                case 1 :  case 2 :  case 3 :  
                                                        sPhase = "小雪"; break; 
                                                case 4 :  case 5 :  case 6 :  
                                                        sPhase = "雪";   break; 
                                                case 7 :  case 8 :  case 9 :   
                                                        sPhase = "大雪"; break; 
                                                case 10 :   
                                                        sPhase = "暴雪"; break; 
                                        } 
                        } 
                        else  
                        { 
                                // 根據雲層判斷晴天陰天 
                                switch(iNewCould) 
                                { 
                                        case 0 :  case 1 :  case 2 :  case 3 :  
                                                sPhase = "晴";   break; 
                                        case 4 :  case 5 :  case 6 :  
                                                sPhase = "少雲"; break; 
                                        case 7 :  case 8 :  case 9 :   
                                                sPhase = "多雲"; break; 
                                        case 10 :   
                                                sPhase = "陰";   break; 
                                } 
 
                                // 晚秋和冬天的早晨有霧 
                                if( iNewCould <= 6 && iHour >= 3 && iHour <= 8 && iSeason != 0 ) 
                                { 
                                        sPhase = "霧"; 
                                        iNewTime = 8 - iHour + 1; 
                                } 
 
                                // 冬天晚上零度以下有霜 
                                if( iNewT < 0 && ( iHour >= 20 || iHour <= 4 ) ) 
                                { 
                                        sPhase = "霜"; 
                                        iNewTime = ( 28 - iHour ) % 24 + 1; 
                                } 
                         
                                // 日出日落天氣晴朗有霞 
                                if( iNewCould <= 6  
                                &&      ( iHour >= 6 && iHour <= 8 || iHour >= 17 && iHour <= 20 ) ) 
                                { 
                                        sPhase = "霞"; 
                                        if( iHour <= 8 ) iNewTime = 8 - iHour + 1; 
                                        else iNewTime = 20 - iHour + 1; 
                                } 
                        } 
 
                        // 設置更新當前天氣狀態 
                        mpCliPhase[sCliPhase]["climate"] = sPhase; 
                        mpCliPhase[sCliPhase]["time"] = iNewTime; 
                        mpCliPhase[sCliPhase]["temperature"] = iNewT; 
                        mpCliPhase[sCliPhase]["cloud"] = iNewCould; 
                        mpCliPhase[sCliPhase]["water"] = iNewWater; 
                        mpCliPhase[sCliPhase]["wind"] = iNewWind; 
 
                        show_nature_phase(sCliPhase); 
                } 
 
        call_out("update_nature_phase", 60  - get_game_minute() ); 
}
