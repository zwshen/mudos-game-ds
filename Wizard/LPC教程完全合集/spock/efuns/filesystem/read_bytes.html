read_bytes(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 read_bytes() - 從一個文件或是一個字符串讀取一連串連續的字節
    　　　 (byte).
　語法:
    　 string read_bytes( string path, int start, int length );

    　 字符串 read_bytes( 字符串 path, 整數 start, 整數 length );

　用法:
    　 這個函數讀取 path 指定的文件內容, 從 start　 指定的第幾字節開始讀取到 length 指定長度的數據. 返回的字節為字符串的類型. 注意, start + length　 的大小絕對不可以超過文件的末端, 否則 read_bytes()　 讀取的動作會失敗. 如果不指定第二及第三個參數, 則會讀取整個文件.
　參考:
    　 read_file(3), write_bytes(3)
　翻譯:
