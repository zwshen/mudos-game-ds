write_bytes(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 write_bytes() - 將連續的字節寫入文件.
　語法:
    　 int write_bytes( string path, int start, string series );

    　 整數 write_bytes( 字符串 path, 整數 start, 字符串 series );

　用法:
    　 write_bytes() 以連續的字節, 從文件 path 的第 start 字節開始寫入. 如果失敗則返回 0, 成功返回 1.
　參考:
    　 write_file(3), read_bytes(3)
