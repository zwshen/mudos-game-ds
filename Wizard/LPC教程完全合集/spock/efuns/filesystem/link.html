link(3)　　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 link() - 將一個文件與另一個文件鏈結
　語法:
    　 void link( string original, string reference );

    　 無返回值 link( 字符串 original, 字符串 reference );

　用法:
    　 將文件 reference　 鏈結到文件 original . link()　 會使主控物件 (master object)　 調用 valid_link( original, reference). 如果 valid_link() 返回 0, 則這次的 link() 失敗. 如果 valid_link() 返回 1, 則還要由 rename() 以同樣的參數 (argument) 調用, 判斷是否成功. 如果 rename(), valid_link()　 均返回 1, 則 link() 成功.

    　 注意: link() 函數創造一個強鏈結 (hard link) , 而非符號鏈結 (symbolic link).

　參考:
    　 rm(3), rmdir(3), rename(3), mkdir(3), cp(3)
