stat(3)　　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 stat() - 返回與一個文件或一個目錄相關的信息.
　語法:
    　 mixed stat( string str, int default: 0 );

    　 混合 stat( 字符串 str, 整數 預設值: 0 );

　用法:
    　 如果 str 是一個普通文件的名字 (不是目錄), 則 stat() 會返回一個數組. 此數組含有關於此文件的信息. 數組的格式如下:

    　 ({ file_size, last_time_file_touched, time_object_loaded })
     

    　 ({ 文件名稱, 最後一次的更動日期, time_object_loaded })

    　 如果 str　 是一個目錄, 或是第二個參數為 -1 , 則 stat() 與 get_dir(3) 的作用相同.

　參考:
    　 get_dir(3), file_size(3)
