file_size(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 file_size() - 取得文件的大小.
　語法:
    　 int file_size( string file );

    　 整數 file_size( 字符串 file );

　用法:
    　 file_size() 返回文件 file 的位元數. 返回 -1 代表 file 不存在, 或是您的權限無法讀寫此文件. 返回 -2 則代表 file　 是一個目錄.
　參考:
    　 stat(3), get_dir(3)
　翻譯:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
