generate_source(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 generate_source() - 產生與指定物件相對照的 C 源代碼
　語法:
    　 void generate_source( string file, void | string out_file );

    　 無返回值 generate_source( 字符串 file, 無參數 或 字符串 out_file );

　用法:
    　 generate_source() 調用 LPC -> C 的編譯程序, 產生指定物件的原始碼. 如果沒有指定輸出的文件, 則輸出的結果會放在 SAVE_BINARIES　 的目錄.
　參考:
    　 valid_asm(4), valid_compile_to_c(4)
　翻譯:
    　 Spock @ FF 97.Feb.13.　 (printed 3/16/95)
