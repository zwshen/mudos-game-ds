previous_object(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 previous_object() - 返回調用目前這個函數的物件 (或物件列表).
　語法:
    　 mixed previous_object( int | void );

    　 混合 previous_object( 整數 或 無參數 );

　用法:
    　 返回一個調用目前函數之物件的物件指針. 請注意, 局部函數 (local function)　 的調用不會讓 previous_object()　 設定為目前的物件, 不會改變 previous_object()　 返回的物件. 如果指定一個正整數為參數, 就返回先前第幾個調用的物件. previous_object(0) 如同 previous_object(), previous_object(1) 是 previous object 的 previous_object(), 以此類推.　 previous_object(-1) 則返回所有的 previous object 數組.
　參考:
    　 call_other(3), call_out(3), origin(3)
