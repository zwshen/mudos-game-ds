call_out(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 call_out() - 延遲調用同一個物件中的函數.
　語法:
    　 void call_out( string | function fun, int delay, mixed arg );

    　 無返回值 call_out( 字符串 或 函數 fun, 整數 delay, 混合 arg );

　用法:
    　 設定在 this_object() 中, 延遲 delay 秒調用函數 fun. 而 arg 當作參數傳入 fun 中.

    　 請注意, 你不能在 fun 中調用 write() 或 say(), 因為 this_player() 被設定成 0. 你可以用 tell_object() 代替.

    　 如果在 driver 的 options.h 中定義 THIS_PLAYER_IN_CALL_OUT, 就可以解決這個問題.

　參考:
    　 remove_call_out(3), call_out_info(3)
