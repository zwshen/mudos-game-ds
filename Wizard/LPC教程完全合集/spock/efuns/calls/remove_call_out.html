remove_call_out(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 remove_call_out() - 取消一個延遲中的外部調用 (call_out).
　語法:
    　 int remove_call_out( string fun );

    　 整數 remove_call_out( 字符串 fun );

　用法:
    　 取消目前這個物件延遲調用的外部調用函數 fun. 返回值就是外部調用還剩下多少延遲時間. 如果函數 fun　 已經沒有延遲的外部調用, 就返回 -1.
　參考:
    　 call_out(3), call_out_info(3)
　翻譯:
    　 Spock @ FF 97.May.27.　 (printed 3/16/95)
