query_verb(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 query_verb() - 返回現在執行的命令名稱.
　語法:
    　 string query_verb( void );

    　 字符串 query_verb( 無參數 );

　用法:
    　 返回目前執行的命令名稱. 如果現在沒有執行任何命令, 則返回 0. 這個函數的用途是: 當好幾個命令會執行同一個函數時, 判斷到底是哪一個命令調用此函數.
　參考:
    　 add_action(3)
　翻譯:
