commands(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 commands() - 返回一些關於此使用者可以使用的命令信息
　語法:
    　 mixed *commands( void );

    　 混合 *commands( 無參數 );

　用法:
    　 返回一個包含四項數據的數組 (array), 說明 this_object() 可以使用的動作 (action) . 第一項數據是命令名稱 ( 即 add_action()　 的命令) ; 而第二項是命令的標誌 (flags) (即 add_action() 的第三個參數, 預設值為 0　 );　 第三項數據是定義此動作的物件. 第四項數據是被此動作調用的函數 (function) (此項數據如果是個函數指針 [function pointer] 則返回 "<function>").
　參考:
    　 add_action(3), enable_commands(3), disable_commands(3)
　翻譯:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
