notify_fail(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 notify_fail() - 返回一段錯誤消息.
　語法:
    　 int notify_fail( string | function str );

    　 整數 notify_fail( 字符串 或 函數指針 str );

　用法:
    　 以 str　 代替系統在命令無效時返回的錯誤消息 :「什麼 ?」當 add_action() 設定的動作返回 0　 時, 就會顯示此段 str 的消息. 使用 notify_fail()　 顯示錯誤消息是一個比較好的方法, 因為此函數允許其他的物件能對同一個命令作出反應. 別用 write()　 來顯示一段錯誤消息, 因為這樣做需要返回 1. ( 除非你想在看到「什麼 ?」之後再看到 write() 的結果 ) 如果你返回 1, 其他的物件就沒有機會對使用者的命令作出反應.

    　 如果參數用的是一個函數指針, 則會調用指定的函數. 如果此函數返回一個字符串, 此字符串就會當作錯誤消息. 只要在此函數中正確地設定 this_player(), 就可以使用 write().

    　 如果 notify_fail()　 被調用一次以上, 只有最後一次的調用有效.

    　 這個函數主要是返回比「什麼 ?」要來得清楚的錯誤消息.

　返回值:
    　 notify_fail() 永遠返回 0.
