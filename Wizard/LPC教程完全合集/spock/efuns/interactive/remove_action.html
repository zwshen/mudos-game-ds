remove_action(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 remove_action - 從局部函數 (local function) 中解除一個命令動詞 (command verb)
　語法:
    　 int remove_action( string fun, string cmd );

    　 整數 remove_action( 字符串 fun, 字符串 cmd );

　用法:
    　 remove_action(3) 從一個物件的局部函數中解除一個命令動詞.
    　 基本上, remove_action() 是相對於 add_action(3) 和 add_verb(3) 的函數. 當您不需要一個動詞時, 就可以用 remove_action() 解除.
　返回值:
    　 remove_action() 的返回值:

    　 1 解除動詞成功.

    　 0 解除失敗.

　參考:
    　 add_action(3), query_verb(3), init(4)
　翻譯:
    　 Spock @ FF 97.May.27.　 (printed 3/16/95)
