resolve(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 resolve() - 轉換一個網際網路地址 (internat address) 為網域名稱 (domain name), 反之亦然..
　語法:
    　 int resolve( string address, string callback_func );

    　 整數 resolve( 字符串 address, 字符串 callback_func );

　用法:
    　 resolve() 會轉換像 "140.128.136.11" 形式的 address　 成為 "bbs.csmc.edu.tw" 這種形式的網域名字. 或是把 "bbs.csmc.edu.tw" 這種形式的 address　 轉換成 "140.128.136.11"　 形式的地址. 轉換成功後, 會調用目前物件的 callback_func. callback_func 的格式為:

    　 void callback(string address, string resolved, int key);

    　 無返回值 callback( 字符串 address, 字符串 resolved, 整數 key );

    　 key 是 resolve() 的返回值. address 是轉換後的地址或網域. 如果轉換失敗, 則會返回 0.

　參考:
    　 query_host_name(3), socket_address(3), query_ip_name(3), query_ip_number(3)
