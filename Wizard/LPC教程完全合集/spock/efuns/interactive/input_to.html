input_to(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 input_to() - 讓下一行輸入的內容送往一個指定的函數
　語法:
    　 varargs void input_to( string | function fun, int flag, ... );

    　 不定參數 無返回值 input_to( 字符串 或 函數指針 fun, 整數 flag, ... );

　用法:
    　 讓使用者下一行輸入的內容作為函數指針 fun 的參數. 輸入的內容不會被驅動程序 (driver) 分析 (parse).

    　 請注意, input_to　 是「非阻隔性」的, 意即調用 input_to 的物件不會為此暫停下來等待輸入. 物件會繼續執行 input_to　 之後的任何程序語句. 在使用者尚未輸入以前, 不會調用函數指針 fun.

    　 如果同一個物件調用 input_to() 超過一次以上, 只有第一次調用有效.

    　 如果參數 flag 為非零值, 則玩家輸入的內容不會回應 (echo) 在畫面上, 也不會出現在監聽 (snoop)　 的畫面上 (在輸入密碼時很有用).

    　 使用者輸入的一行內容會被當成調用函數指針 fun　 時的第一個參數 (類型為字符串).　 在 flag 之後的參數會當成 fun　 的第二個、第三個......等等的參數.

　參考:
    　 call_other(3), call_out(3), get_char(3)
