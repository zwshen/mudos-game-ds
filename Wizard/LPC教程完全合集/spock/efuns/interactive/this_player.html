this_player(3)　 MudOS v21c2　 (5 Sep 1994)

　語法:
    　 object this_player( int flag );

    　 物件 this_player( 整數 flag );

　用法:
    　 返回調用此函數的玩家 (player) 物件. 請注意, 即使是從玩家物件內部使用 this_player(), 其返回值也可能與 this_object() 不同. 如果以 this_player(1) 調用, 則返回調用此函數的可互動 (interactive)　 物件. this_player(1)　 的返回值在某些情形下與 this_player() 不同. 例如: 系統管理員 (admin)　 使用 command()　 強制一名玩家做出一些動作時.
　參考:
    　 this_object(3)
