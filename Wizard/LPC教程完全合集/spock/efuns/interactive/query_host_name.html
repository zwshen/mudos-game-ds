query_host_name(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 query_host_name() - 返回使用的主機名稱 (host name)
　語法:
    　 string query_host_name( void );

    　 字符串 query_host_name( 無參數 );

　用法:
    　 query_host_name() 返回使用的主機名稱.
　返回值:
    　 如果查詢主機成功, 則返回一個字符串.

    　 如果失敗, 則返回一個空字符串.

　參考:
    　 resolve(3), socket_address(3), query_ip_name(3), query_ip_number(3)
　翻譯:
    　 Spock @ FF 97.Apr.30.　 (printed 3/16/95)
