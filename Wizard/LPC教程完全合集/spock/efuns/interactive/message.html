message(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 message() - 送消息給活著的 (living) 物件.
　語法:
    　 void message( mixed class, string message, mixed target, mixed exclude );

    　 無返回值 message( 混合 class, 字符串 message, 混合 target, 混合 exclude );

　用法:
    　 message() 調用目標名單中所有物件 (除了 exclude 名單以外) 的 receive_message( 混合 class, 字符串 message ) 函數. 這樣基本上只是告訴此物件一個消息.
    　
    　 class 是此段消息的種類, 用於處理消息的介面之類的地方. 例如 combat, shout, emergency 等等.

    　 message 是欲送出的消息字符串.

    　 target 是欲送達消息的物件名單, 可以是一個物件或物件指針 (pointer). 如果目標是一個非活著的 (non-living) 的物件, 在此物件內的所有物件都會收到消息.

    　 exclude 是不應收到此消息的物件名單, 可以是一個物件或物件數組 (array).

　參考:
    　 say(3), write(3), shout(3), tell_object(3), tell_room(3)
