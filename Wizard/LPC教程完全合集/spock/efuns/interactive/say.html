say(3)　　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 say() - 把消息傳遞給所有在同一個環境中的使用者.
　語法:
    　 varargs void say( string str, object obj );

    　 不定參數 無返回值 say( 字符串 str, 物件 obj );

　用法:
    　 把消息傳遞給位於跟發出消息者同一環境的物體, 以及所有在發出消息者裡面的物體. 發出消息者只能是 this_player(), 或是　 this_player() == 0 的 this_object().

    　 第二個參數可有可無. 如果指定了第二個參數 obj, 則消息不會傳遞給 obj. 如果 obj 不是一個物件, 而是一個物件數組 (an array of objects) , 則消息就不會傳遞給數組內的物件.

　參考:
    　 message(3), write(3), shout(3), tell_object(3), tell_room(3)
　翻譯:
