find_player(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 find_player() - 用名字尋找一個玩家
　語法:
    　 object find_player( string str );

    　 物件 find_player( 字符串 str );

　用法:
    　 與 find_living()　 相似, 但是只尋找可互動的 (interactive) 或是曾經為可互動的物件.
　參考:
    　 lower_case(3), find_living(3), livings(3), users(3), set_living_name(3)
　翻譯:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
