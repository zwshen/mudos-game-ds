receive(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 receive() - 對目前的物件顯示一段消息.
　語法:
    　 int receive( string message );

    　 整數 receive( 字符串 message );

　用法:
    　 這個外部函數 (efun) 是作為 mud　 驅動程序 (driver) 的 add_message() 函數的介面. 這個函數設計的目的是對目前物件顯示消息. 如果目前的物件是「可互動的」則返回 1. 不是則返回 0. receive() 應被 catch_tell(4) 或是 receive_message(4) 調用.
　參考:
    　 catch_tell(4), receive_message(4), message(3)
　翻譯:
    　 Spock @ FF 97.May.24.　 (printed 3/16/95)
