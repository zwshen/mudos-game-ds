command(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 command() - 如同該物件般, 執行一個命令.
　語法:
    　 int command( string str, object ob );

    　 整數 command( 字符串 str, 物件 ob );

　用法:
    　 讓物件 ob 執行 str　 的命令. 如果不指定 ob , 則視同為 this_object().　 請注意, 第二個參數可以使用的範圍是由該 MUD　 的系統管理員所決定, 所以您經常無法使用您所指定的物件動作. 如果該物件無法執行命令, 函數則會返回 0 值. 如果可以執行, 則會返回一個數字. 此數字是 LPC　 執行的該命令的「執行花費時間」 (evaluation cost). 數字越大則此命令就越費時, 但是這個數字並不夠精確, 其所使用的量度 (scale)　 是主觀 (subjective) 而不可靠 (unreliable)　 的.
　參考:
    　 add_action(3), enable_commands(3)
　翻譯:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
