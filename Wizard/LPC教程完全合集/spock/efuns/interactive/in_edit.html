in_edit(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 in_edit() - 判斷一個玩家是否在文書編輯程序中.
　語法:
    　 string in_edit( object default : F_THIS_OBJECT );

    　 字符串 in_edit( 物件 預設值 : F_THIS_OBJECT );

　用法:
    　 如果一個指定的物件正在文書編輯工作中, 就返回被編輯文件名稱. 如果不是, 就返回 0.
　參考:
    　 ed(3), in_input(3)
　翻譯:
    　 Spock @ FF 97.Feb.14.　 (printed 3/16/95)
