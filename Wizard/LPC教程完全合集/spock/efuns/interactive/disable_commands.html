disable_commands(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 disable_commands() - 設定一個活著的 (living) 物件為「非活著」(non-living)
　語法:
    　 int disable_commands( void );

    　 整數 disable_commands( 無參數 );

　用法:
    　 讓一個活著的物件變成「非活著」(non-living). 所謂「非活著」, 就是讓物件變成以下的狀態:

    　 add_actions 失效.
    　 livingp 函數返回「偽」值 (false, 0).
    　 如果此物件是「可互動的」 (interactive), 就讓這個使用者
    　 無法輸入命令 (input_to 函數例外).

    　 disable_commands 永遠返回 0 值.

　參考:
    　 enable_commands(3)
