socket_listen(3)　 MudOS v21c2　　 (5 Sep 1994)

　名稱：
    　 socket_listen() - 傾聽(listen)一個 socket 的連接(connection)。
　語法：
    　 #include <socket_err.h>

    　 int socket_listen( int s, string listen_callback );

    　 整數 socket_listen( 整數 s, 字符串 listen_callback );

　用法：
    　 為了要接受連接，首先要以 socket_create(3) 建立一個 socket，再以 socket_listen(3) 將這個 socket 放到傾聽模式，最後才以 socket_accept(3) 接受這個連接。socket_listen() 調用只能作用於 STREAM 或是 MUD 類型的 sockets。

    　 listen_callback 參數是給驅動程序在得到正傾聽中 socket 的連接時調用的函數名稱。這個傾聽回呼(callback)函數應該是這個格式：
     

      　　 void listen_callback(int fd)

    　 這裡的 fd 是正傾聽中的 socket。

　返回值：
    　 socket_listen() 返回：
     
      　　 成功時是 EESUCCESS。

      　　 失敗時是一個含有如下意義的負值。

　錯誤碼：
    　 EEFDRANGE　　　　　 描述值(descriptor)超出範圍。

    　 EEBADF　　　　　　　　 無效的描述值。

    　 EESECURITY　　　　 企圖違反安全。

    　 EEMODENOTSUPP　 不支援此 socket 類型。

    　 EENOADDR　　　　　　 socket 尚未設定位址。

    　 EEISCONN　　　　　　 socket 已連接。

    　 EELISTEN　　　　　　 傾聽問題。

　參照：
    　 socket_accept(3), socket_connect(3), socket_create(3)
