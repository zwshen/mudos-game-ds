socket_write(3)　 MudOS v21c2　　 (5 Sep 1994)

　名稱：
    　 socket_write() - 通過一個 socket 傳送數據
　語法：
    　 #include <socket_err.h>

    　 int socket_write( int s, mixed message, void | string address );

    　 整數 socket_write( 整數 s, 混合 message, 無參數 | 字符串 address );

　用法：
    　 socket_write() 用一個 socket s 來傳送數據。如果 socket s 是 STREAM 或是 MUD 類型，這個 socket 就必須是已連接的(connected) 而且不能指定位址。如果 socket 是 DATAGRAM 類型，就必須指定位址。位址的格式是 "127.0.0.1 23"。
　返回值：
    　 socket_write() 返回：
     
      　　 成功時是 EESUCCESS。

      　　 失敗時是一個含有下述意義的負值。

　錯誤值：
    　 EEFDRANGE　　　　　 描述值(descriptor)超出範圍。

    　 EEBADF　　　　　　　　 無效的描述值。

    　 EESECURITY　　　　 企圖違反安全。

    　 EENOADDR　　　　　　 socket 未指定位址。

    　 EEBADADDR　　　　　 位址格式問題。

    　 EENOTCONN　　　　　 socket 尚未連接。

    　 EEALREADY　　　　　 操作已在進行中。

    　 EETYPENOTSUPP　 不支援此物件類型。

    　 EEBADDATA　　　　　 傳送數據含有太多層次。

    　 EESENDTO　　　　　　 sendto 問題。

    　 EEMODENOTSUPP　 不支援此 socket 類型。

    　 EEWOULDBLOCK　　 操作將會停滯(blocked)。

    　 EESEND　　　　　　　　 傳送問題。

    　 EECALLBACK　　　　 等待回呼(callback)中。

　參照：
