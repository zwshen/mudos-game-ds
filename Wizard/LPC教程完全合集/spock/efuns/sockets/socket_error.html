socket_error(3)　 MudOS v21c2　　 (5 Sep 1994)

　名稱：
    　 socket_error() - 返回一個 socket 錯誤碼的文字語句
　語法：
    　 #include <socket_err.h>

    　 string socket_error( int error );

    　 字符串 socket_error( 整數 error );

　用法：
    　 socket_error() 返回一個描述所產生之錯誤的字符串。
　返回值：
    　 socket_error() 返回：
     
      　 成功時是一個描述錯誤碼的字符串。

      　 錯誤的輸入值時是 "socket_error: invalid error number"。
      　　　　　　　　　　　　　　 （譯註："socket_error: 無效的錯誤碼"）

　參照：
    　 socket_create(3), socket_connect(3)
