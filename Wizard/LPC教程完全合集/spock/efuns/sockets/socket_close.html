socket_close(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱：
    　 socket_close() - 關閉一個 socket
　語法：
    　 #include <socket_err.h>

    　 int socket_close( int s );

    　 整數 socket_close( 整數 s );

　用法：
    　 socket_close() 會關閉 socket s。這會釋放出一個被使用的 socket efun 插槽(slot)。
　返回值：
    　 socket_close() 返回：
     
      　　 成功時是 EESUCCESS。

      　　 失敗時是含如下意義的負值。

　錯誤值：
    　 EEFDRANGE　　　　　 描述值(descriptor)超出範圍。

    　 EEBADF　　　　　　　　 無效的描述值。

    　 EESECURITY　　　　 企圖違反安全。

　參照：
    　 socket_accept(3), socket_create(3)
