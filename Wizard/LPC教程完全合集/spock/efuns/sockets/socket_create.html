socket_create(3)　 MudOS v21c2　　 (5 Sep 1994)

　名稱：
    　 socket_create() - 建立一個 efun socket
　語法：
    　 #include <socket_err.h>

    　 int socket_create( int mode, string read_callback, void | string close_callback );

    　 整數 socket_create( 整數 mode, 字符串 read_callback, 無參數 | 字符串 close_callback );

　用法：
    　 socket_create() 建立一個 efun socket。mode 決定了要建立哪種類型的 socket。目前支援的 socket 類型有：

    　 MUD　　　　　　　　 以 TCP 協議傳送 LPC 數據。

    　 STREAM　　　　　 以 TCP 協議傳送原始數據。

    　 DATAGRAM　　　 以 UDP 協議傳送數據。

    　 read_callback 這個參數是給驅動程序在 socket 由對方接到數據時調用用的函數名稱。這個讀取回呼(callback)函數應該是如下格式：
     

      　　 void read_callback(int fd, mixed message)

    　 這裡的 fd 是接收數據的 socket，而 message 是接收到的數據。

    　 close_callback 這個參數是給驅動程序在 socket 不正常關閉時調用的函數名稱，如不同於 socket_close() 的調用結果。這個關閉回呼函數應該是如下格式：
     

      　　 void close_callback(int fd)

    　 這裡的 fd 是已被開閉的 socket。注意：close_callback 不能用在 DATAGRAM 類型的 socket 上。

　返回值：
    　 socket_create() 返回：
     
      　　 成功時是一個非負的描述值(descriptor)。

      　　 失敗時是一個下列意義的負值。

　錯誤值：
    　 EEMODENOTSUPP　 不支援此 socket 類型。

    　 EESOCKET　　　　　　 建立 socket 的問題。

    　 EESETSOCKOPT　　 setsockopt 的問題。

    　 EENONBLOCK　　　　 設定非停滯(non-blocking)模式的問題。

    　 EENOSOCKS　　　　　 沒有多餘的 efun sockets。

    　 EESECURITY　　　　 企圖違反安全。
