socket_connect(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱：
    　 socket_connect() - 初始化一個 socket 上的連接(connection)
　語法：
    　 #include <socket_err.h>

    　 int socket_connect( int s, string address, string read_callback, string write_callback );

    　 整數 socket_connect( 整數 s, 字符串 address, 字符串 read_callback, 字符串 write_callback );

　用法：
    　 參數 s 是一個 socket。s 必須是一個 STREAM 類型或是一個 MUD 類型的 socket。address 是這個 socket 將要嘗試連接的位址。位址是 "127.0.0.1 23" 這種格式。

    　 read_callback 參數是給驅動程序在由它的另一端得到數據時調用的函數名稱。這個讀取回呼(callback)函數應該是如下格式：
     

      　　 void read_callback(int fd, mixed message)

    　 這裡的 fd 是要接收數據的 socket，而 message 是接收到的數據。

    　 write_callback 參數是給驅動程序在 socket 已準備好寫入時調用的函數名稱。這個寫入回呼函數應該是如下格式：
     

      　　 void write_callback(int fd)

    　 這裡的 fd 是已準備好寫入的 socket。

　返回值：
    　 socket_connect() 返回：
     
      　　 成功時是 EESUCCESS。

      　　 失敗時是一個含有如下意義的負值。

　錯誤值：
    　 EEFDRANGE　　　　　 描述值(descriptor)超出範圍。

    　 EEBADF　　　　　　　　 無效的描述值。

    　 EESECURITY　　　　 企圖違反安全。

    　 EEMODENOTSUPP　 不支援此 socket 類型。

    　 EEISLISTEN　　　　 socket 正在傾聽(listening)。

    　 EEISCONN　　　　　　 socket 已連接。

    　 EEBADADDR　　　　　 位址格式問題。

    　 EEINTR　　　　　　　　 中斷的系統調用。

    　 EEADDRINUSE　　　 位址已被使用。

    　 EEALREADY　　　　　 操作已在進行中。

    　 EECONNREFUSED　 連接被拒絕。

    　 EECONNECT　　　　　 連接問題。

　參照：
    　 socket_accept(3), socket_close(3), socket_create(3)
