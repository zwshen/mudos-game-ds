socket_bind(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱：
    　 socket_bind() - 命名(bind)一個 socket
　語法：
    　 #include <socket_err.h>

    　 int socket_bind( int s, int port );

    　 整數 socket_bind( 整數 s, 整數 port );

　用法：
    　 socket_bind() 為一個未命名的 socket 指定一個名字。當一個 socket 由 socket_create(3) 建立後，它已存在於一個命名空間（位址族, address family）但還沒有指定名字。socket_bind() 需要埠號來為 socket s 命名。
　返回值：
    　 socket_bind() 返回：
     
      　　 成功時是 EESUCCESS。

      　　 失敗時是一個有如下意義的負值。

　錯誤值：
    　 EEFDRANGE　　　　　 描述值(descriptor)超出範圍。

    　 EEBADF　　　　　　　　 無效的描述值。

    　 EESECURITY　　　　 企圖違反安全。

    　 EEISBOUND　　　　　 socket 已命名。

    　 EEADDRINUSE　　　 此位址已被使用。

    　 EEBIND　　　　　　　　 命名問題。

    　 EEGETSOCKNAME　 getsockname 問題。

　參照：
    　 socket_connect(3), socket_create(3), socket_listen(3)
