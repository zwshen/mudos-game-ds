random(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 random() - 返回一個「假性」亂數 (pseudo-random number).
　語法:
    　 int random( int n );

    　 整數 random( 整數 n );

　用法:
    　 返回一個範圍從 0　 到 n - 1　 的假性亂數.
　翻譯:
    　 Spock @ FF 97.May.2.　 (printed 3/16/95)
　譯按:
    　 真正的亂數必須先給予一個種子數 (seed number), 再藉由查亂數表取得一組亂數, 而這些亂數值均介於 0 與 1　 之間.
