function_profile(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 function_profile() -　 對一個物件取得函數的側寫 (profiling) 信息.
　語法:
    　 mixed *function_profile( object ob );

    　 混合 *function_profile( 物件 ob );

　用法:
    　 返回對 ob 或是 this_object() (如果不予指定 ob ) 的函數側寫信息. 這個函數只有在驅動程序 (driver) 編譯時有定義 PROFILE_FUNCTIONS 才有效.
　返回值:
    　 此函數返回一個映射變量的數組, 每一個在 ob 中的函數返回的映射格式如下:
    　
      　 ([　　　 "name"　　　　 : 函數的名稱,
        　　　　 "calls"　　　 : 調用的次數,
    　
      　 /* cpu 時間以微秒 (microsecond) 為單位 */
      　　　 "self"　　　　 : cpu_time_spent_in self,
      　　　 "children" : cpu_time_spent_in_children
      　　 ])
    　 此處的用途與 CPU 時脈有關, 雖然單位為微秒 (microseconds), 但實際上的精確度要差得多.
　參考:
    　 rusage(3), time_expression(3), opcprof(3)
