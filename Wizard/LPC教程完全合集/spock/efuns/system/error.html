error(3)　 MudOS v21c2　 error(3)

　名稱:
    　 error - 產生一個執行時段 (run_time) 錯誤.
　語法:
    　 void error( string err );

    　 無返回值 error( 字符串 err );

　用法:
    　 調用 error() 時, 會產生一個執行時段錯誤 err. 並中止目前正在執行的執行緒脈絡 (thread). 而發生錯誤的程序會被追蹤錯誤的出處, 並紀錄在除錯紀錄 (debug log) 的文件中.
　參考:
    　 catch(3), throw(3), error_handler(4)
