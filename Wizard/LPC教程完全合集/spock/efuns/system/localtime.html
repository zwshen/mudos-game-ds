localtime(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 localtime() - 轉換成本地時間
　語法:
    　 #include <localtime.h>

    　 mixed *localtime( int time );

    　 混合 *localtime( 整數 time );

　用法:
    　 localtime() 轉換一個整數的時間值 (像是 time() 返回值) 成為一個數組. 這個數組就是目前的本地時間. 在以前 time() 是函數用來取得格林威治時間 (GMT (UTC)), 而本地時間的定義是與格林威治時間的差值. 現在這種換算的方法已經沒有必要了. localtime() 返回秒數、分鐘、小時、日期、月數、年數、星期幾、一年中的第幾天、本地的時區名稱、本地時區與格林威治時區的差值. 這些時間的信息直接由作業系統取得, 而且使驅動程序不需要用到 MUD　 本身的設定文件.

    　 localtime() 返回一個包含上述時間信息的數組. 每一個時間值的索引關鍵字 (index)　 由 localtime.h　 定義. 下表則整理出 localtime()　 返回的數組內容.
     

      　 int LT_SEC　 秒數 (0..59)
      　 int LT_MIN　 分鐘 (0..59)
      　 int LT_HOUR　 小時 (0..23)
      　 int LT_MDAY　 日數 (1..31)
      　 int LT_MON　 月數 (0..11)
      　 int LT_YEAR　 年份 (必須大於 1900)
      　 int LT_WDAY　 星期幾 (0..6)
      　 int LT_YDAY　 一年中的第幾天 (0..365)
      　 int LT_GMTOFF　 格林威治時間相差的秒數 (UTC)
      　 string LT_ZONE　 本地時區的名稱
　參考:
    　 ctime(3), time(3), time_expression(3), uptime(3), /include/localtime.h
