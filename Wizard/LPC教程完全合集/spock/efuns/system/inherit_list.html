inherit_list(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 inherit_list() - 取得一個物件所繼承的父物件 (parents) 清單.
　語法:
    　 string *inherit_list( object obj );

    　 字符串 *inherit_list( 物件 obj );

　用法:
    　 返回物件 obj　 所繼承之父物件的文件名稱, 以字符串數組 (array) 的形式返回. 例如: A 繼承 B, 而 B　 繼承 C　 , inherit_list(A) 會返回一個含有 B 的數組, 而沒有 C.
　參考:
    　 deep_inherit_list(3), inherits(3)
　翻譯:
    　 Spock @ FF 97.Feb.14.　 (printed 3/16/95)
