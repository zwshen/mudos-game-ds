uptime(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 uptime() - 返回上一次驅動程序 (driver) 啟動至今的秒數.
　語法:
    　 int uptime( void );

    　 整數 uptime( 無參數 );

　用法:
    　 uptime() 返回上一次驅動程序重新啟動至今有幾秒.
　參考:
    　 time(3), ctime(3), localtime(3), time_expression(3)
　翻譯:
    　 Spock @ FF 97.jul.25.　 (printed 3/16/95)
