set_reset(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 set_reset - 更改一個物件距離下次重新設定 (reset) 的時間.
　語法:
    　 varargs void set_reset( object ob, int time );

    　 不定參數 無返回值 set_reset( 物件 ob, 整數 time );

　用法:
    　 設定物件 ob 距離下次重新設定的時間為整數 time 指定的秒數. 如果不指定 time, 則使用驅動程序普通設定的公式.
     
　 公式:
    　　 reset time = current_time + reset_time / 2 + random(reset_time / 2)
　參考:
    　 reset(4)
