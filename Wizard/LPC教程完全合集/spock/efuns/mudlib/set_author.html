set_author(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 set_author - 設定與一個物件相關的作者.
　語法:
    　 void set_author( string author );

    　 無返回值 set_author( 字符串 author );

　用法:
    　 為了要統計作者和區域 (domain) 的狀況, 所有的物件都有相關的作者和區域. 區域只能由主控物件 (master object) 的 domain_file(4) 函數設定. 作者可以由主控物件的 author_file(4) 設定一些預設值 (default value),　 也可以使用 set_author() 函數來更改.

    　 調用 set_author() 的物件, 其相關的作者就會經由 set_author()　 更改之. 這個設定的作者會影響以後 mud 函數庫 (mudlib) 的統計數字. 結果可能會導致一些統計項目出現奇怪的數字, 例如: 物件 (objects)　 和數組大小 (array_size)　 兩項. 因為這個物件可能有初始的數組 (initialized arrays), 或由原始的作者創造出來. 但是可能中途用 set_author() 改成其他的作者, 又摧毀此物件或是釋放數組的空間. 這樣一來, 統計數字上減去的項目就不是原來作者的值, 而是另一個作者的.

    　 要解決這個問題, 只可以在 create(4) 中使用 set_author(), 而且盡量用在任何數組聲明之前.

　參考:
    　 author_file(4), domain_file(4), author_stats(3), domain_stats(3)
