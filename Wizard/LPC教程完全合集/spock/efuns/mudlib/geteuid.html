geteuid(3) MudOS v21c2 (5 Sep 1994) geteuid(3)

　名稱:
    　 geteuid() - 返回一個物件或是一個函數的有效使用者識別名稱 (effective user id, euid)
　語法:
    　 string geteuid( object | function );

    　 字符串 geteuid( 物件 或 函數指針 );

　用法:
    　 如果參數是一個物件, geteuid 會返回物件的有效使用者識別名稱 (euid) . 如果指定的參數是一個函數指針 (function pointer,　 變量類型為 function), 則會返回創造這個函數指針之物件的 euid . 如果這個函數指針的變量, 在此物件創造時沒有 euid,　 則會使用物件的使用者識別名稱 (user id, uid) 代替.
　參考:
    　 seteuid(3), getuid(3), functionp(3), export_uid(3), previous_object(3), this_object(3), valid_seteuid(4)
