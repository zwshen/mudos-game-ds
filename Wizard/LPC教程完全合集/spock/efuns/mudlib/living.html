living(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 living() - 判斷一個指定的物件是否「活著」
　語法:
    　 int living( object ob );

    　 整數 living( 物件 ob );

　用法:
    　 如果 ob 是一個活著的物件則返回真值 (true, 即非零值).　 (也就是說, ob 曾經調用過 enable_commands() )
　參考:
    　 interactive(3), enable_commands(3)
　翻譯:
    　 Spock @ FF 97.Feb.18.　 (printed 3/16/95)
