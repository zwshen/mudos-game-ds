export_uid(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 export_uid() -　 設定一個物件的使用者識別名稱 (user id, uid).
　用法:
    　 int export_uid( object ob );

    　 整數 export_uid( 物件 ob );

　用法:
    　 設定 ob 的使用者識別名稱 (uid) 為 this_object() 的有效使用者識別名稱 (effective user id, effective uid). 這個函數只有在 ob 的有效使用者識別名稱為 0 時有效.
　參考:
    　 this_object(3), seteuid(3), getuid(3), geteuid(3), previous_object(3), valid_seteuid(4)
