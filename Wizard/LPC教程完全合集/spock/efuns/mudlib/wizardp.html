wizardp(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 wizardp() - 判斷指定的物件是否執行過 enable_wizard()
　語法:
    　 int wizardp( object );

    　 整數 wizardp( 物件 );

　用法:
    　 如果 arg　 曾執行過 enable_wizard(), wizardp() 就返回 1.
　參考:
    　 disable_wizard(3), enable_wizard(3)
　翻譯:
    　 Spock @ FF 97.jul.25.　 (printed 3/16/95)
