livings(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 livings() - 返回所有活著 (living) 物件的一個數組
　語法:
    　 object *livings( void );

    　 物件 *livings( 無參數 );

　用法:
    　 返回指向所有活著物件之指針 (pointer)　 的一個數組.
    　 活著的物件是指曾經調用過 enable_commands()　 的物件.
　參考:
    　 enable_commands(3), find_living(3), users(3), objects(3)
　翻譯:
    　 Spock @ FF 97.Feb.18.　 (printed 3/16/95)
