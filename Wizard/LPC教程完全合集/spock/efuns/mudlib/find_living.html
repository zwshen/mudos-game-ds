find_living(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 find_living() - 找出一個指定識別名稱 (id) 的活 (living)　 物件
　語法:
    　 object find_living( string str );

    　 物件 find_living( 字符串 str );

　用法:
    　 首先找標記為活著的 (living) , 並符合識別名稱 (id) 為 str 的物件. 已經調用過 enable_commands()　 的物件才被視為活著的物件. 並且此物件必須以 set_living_name() 設定名稱, 此一名稱才會放進搜尋表 (hash table) 中以提升尋找活物件的速度.
　參考:
    　 living(3), livings(3), find_player(3), users(3), enable_commands(3), set_living_name(3)
　翻譯:
    　 Spock @ FF 97.Feb.12.　 (printed 3/16/95)
