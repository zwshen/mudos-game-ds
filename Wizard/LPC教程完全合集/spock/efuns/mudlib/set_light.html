set_light(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 set_light() - 更改或查詢一個物件的照明程度 (light level).
　語法:
    　 int set_light( int light_level_adjustment );

    　 整數 set_light( 整數 light_level_adjustment );

　用法:
    　 將 light_level_adjustment 指定為 0, 則查詢物件目前的照明程度. 指定為正整數則增加照明的程度, 負整數降低.
　翻譯:
