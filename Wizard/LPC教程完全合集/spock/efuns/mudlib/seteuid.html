seteuid(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 seteuid() - 設定一個物件的有效使用者識別名稱 (effective user id, euid)
　語法:
    　 int seteuid( string str );

    　 整數 seteuid( 字符串 str );

　用法:
    　 設定有效使用者識別名稱為 str　 的內容. 在 master.c 中的 valid_seteuid() 函數決定著一個物件的有效使用者識別名稱可以設定為何.

    　 當一個物件的有效使用者識別名稱為 0, 則這個物件的使用者識別名稱 (uid)　 只能用 export_uid() 更改.

    　 同時, 這個物件不能載入或是復制任何物件.

　參考:
    　 export_uid(3), getuid(3), geteuid(3), this_object(3), valid_seteuid(4)
