read_buffer(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 read_buffer() - 以緩衝區數據 (buffer) 的類型返回文件的內容, 或是以字符串類型 (string) 返回緩衝區一部份的數據.
　語法:
    　 string | buffer read_buffer( string | buffer src, int start,　 int len );

    　 字符串 或 緩衝區 read_buffer( 字符串 或 緩衝區 src, 整數 start, 整數 len );

　用法:
    　 如果 src　 是一個字符串 (文件名稱) , 就會從 start　 指定的第幾字節 (byte) 開始, 讀取 len　 指定長度的文件內容. 讀取的內 容以緩衝區的數據類型返回. 如果不指定 start 和 len , 則會讀取整個文件的內容.

    　 如果 src　 是一個緩衝區, 就從 start　 指定的第幾位元開始讀取 len 長度的內容. 然後以字符串的類型返回.

    　 請注意, 你可以從一個文件讀進緩衝區的數據, 受到編譯時段組態文件 (runtime config file)　 中定義的 maximum byte transfer (最大字節傳輸量) 控制.

　參考:
    　 write_buffer(3), allocate_buffer(3), bufferp(3), read_bytes(3), write_bytes(3)
