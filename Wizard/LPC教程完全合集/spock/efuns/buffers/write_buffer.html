write_buffer(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 write_buffer() - 將一個緩衝區 (buffer) 寫入一個文件, 或從某個數據來源讀進一個緩衝區.
　語法:
    　 int write_buffer( string | buffer dest, int start, mixed source );

    　 整數 write_buffer( 字符串 或 緩衝區 dest, 整數 start, 混合 source );

　用法:
    　 如果 dest 是一個文件, 則 source 必須為一個整數 (並以 network-byte-order　 寫入文件) 、一個緩衝區、或一個字符串, 而從文件 dest 的第 start　 字節開始寫入 source 的內容.

    　 如果 dest 是一個緩衝區, 則 source 會從第 start　 字節開始寫入緩衝區. 如果 source 是整數, 則會以 network-byte-order　 的格式寫入.

　參考:
    　 read_buffer(3), allocate_buffer(3)
