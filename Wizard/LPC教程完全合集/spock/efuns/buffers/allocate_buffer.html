allocate_buffer(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 allocate_buffer() - 配置內存給一個緩衝區 (buffer).
　語法:
    　 buffer allocate_buffer( int size );

    　 緩衝區 allocate_buffer( 整數 size );

　用法:
    　 配置一塊有 size 個元素 (elements) 的緩衝區. 元素的數目必須大於 0, 而且不可超過系統設定的上限值 (通常是 10000 左右) . 所有數組元素的初始值 (initialized) 均為 0.
　參考:
    　 bufferp(3), read_buffer(3), write_buffer(3), sizeof(3), to_int(3)
