match_path(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 match_path() - 在一個映射變量 (mapping) 中找尋路徑名稱 (path)
　語法:
    　 mixed match_path( mapping m, string str );

    　 混合 match_path( 映射 m, 字符串 str );

　用法:
    　 match_path() 在一個映射變量中找路徑名. match_path() 會假設所有的關鍵字 (key) 為字符串 (string) 類型. 而內容值 (value) 是啥則無關緊要. 這個外部函數 (efun) 會找出符合條件的最長字符串. 以 / 結尾的關鍵字會找出此關鍵字底下所有的路徑名稱, 意即 / 算是一個萬用字符 (wildcard), 視為此路徑底下的任何東西 (譯按: 相當於文件或路徑名稱用的 * 號).
