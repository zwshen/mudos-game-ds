allocate_mapping(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 allocate_mapping() - 預先配置 (pre-allocate) 一塊內存給一個映射 (mapping).
　語法:
    　 mapping allocate_mapping( int size );

    　 映射 allocate_mapping( 整數 size );

　用法:
    　 返回一個預先配置 size 個元素 (elements) 的映射.
     
　 範例:
      　　 mapping x;
      　　 int y = 200;

      　　 x = allocate_mapping(y);

    　 y 是這個映射的初始大小. 如果您已經知道映射有多大, 使用 allocate_mapping 是比較好的方法 (在範例裡是 200 個元素) , 因為一次就設定好要配置的內存, 就內存的使用上是比較有效率的. 如果您要用映射儲存 200 項數據, 用 allocate_mapping 是較適合而有效率的方式.

    　 注意 !　 如果您預計會刪除映射中的許多項元素, 就應該用 x = ([]) 來初始化範例中的映射 x, 而最好別用 allocate_mapping().

    　 注意: 在此, 實際上映射的大小是沒有意義的.

    　 x = allocate_mapping( 200 ); 等於 x = ([]);
