each(3)　　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 each() - 分次返回一個映射變量 (mapping) 的各個 (key, value) 的內容.
　語法:
    　 mixed *each( mapping m, int reset );
　用法:
    　 each()　 函數把一個映射變量裡的每對 (key, value) 一次一次地返回. each() 返回來的次序與 keys() 和 values() 兩個函數返回來的次序一樣. 當 each() 讀到映射變量的尾端時, 會返回一個空的向量 (null vector, 即 (0,0)). 而在返回空向量之後, 下一個 each() 返回的內容又會從頭開始.

    　 當不同的物件使用 each() 讀取相同的映射變量時, 一定都會從頭開始. 雖然 MudOS 有單執行緒 (single thread) 的性質, 但是也不致於讓 each() 執行時出現問題. ( 實際上一個物件無法於使用 each() 讀取映射函數讀到一半時, 跳開目前讀 (key, value) 的位置)

    　 如果指定第二個參數為 1, 則 each() 會重新從頭開始讀取.
     

　 範例:
     
      　 mixed *pair;

      　 while ((pair = each(x)) != ({})) {
      　　 write("key　　 = " + pair[0] + "\n");
      　　 write("value = " + pair[1] + "\n");
      　 }

　參考:
    　 keys(3), values(3)
