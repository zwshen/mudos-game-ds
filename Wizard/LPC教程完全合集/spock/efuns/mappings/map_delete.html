map_delete(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 map_delete() -　 以關鍵字從一個映射變量中刪除一對 (關鍵字, 內容值) ((key, value)).
　語法:
    　 void map_delete( mapping m, mixed element );

    　 無返回值 map_delete( 映射 m, 混合 element );

　用法:
    　 map_delete 刪除映射變量中關鍵字符合 element　 的 (關鍵字, 內容值) .

    　 範例:
    　

      　 mapping names;

      　 names = ([]);
      　 names["truilkan"] = "john";
      　 names["wayfarer"] = "erik";
      　 names["jacques"] = "dwayne";

    　 則:
    　
      　 map_delete(names,"truilkan");
    　 會使映射變量 names 成為:
    　
      　 (["wayfarer" : "erik", "jacques" : "dwayne"])
    　 在 map_delete( names, "truilkan" ) 之後, key(names) 就不會包含 truilkan (除非後來又用 ("truilkan", *) 把 truilkan 加回映射變量裡)
　參考:
    　 allocate_mapping(3)
