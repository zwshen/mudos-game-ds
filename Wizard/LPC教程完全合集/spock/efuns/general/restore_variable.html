restore_variable(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 restore_variable() - 從一個字符串讀回變量值.
　語法:
    　 mixed restore_variable( string value );

    　 混合 restore_variable( 字符串 value );

　用法:
    　 從一個字符串讀回變量值. 字符串的格式與 save_object() 和 restore_object() 所使用的相同.
　參考:
    　 restore_object(3)
　翻譯:
    　 Spock @ FF 97.Jun.2.　 (printed 3/16/95)
