typeof(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 typeof() - 返回一個式子 (expression) 的數據類型 (type).
　語法:
    　 int typeof( mixed var );

    　 整數 typeof( 混合 var );

　用法:
    　 返回一個式子的數據類型. <typeof.h> 中定義返回值為:
    　
      　 整數　 INT　 2
      　 字符串　 STRING　 4
      　 數組　 ARRAY　 8
      　 物件　 OBJECT　 16
      　 映射　 MAPPING　 32
      　 函數指針 FUNCTION 64
      　 浮點數　 FLOAT　 128
      　 緩衝區　 BUFFER　 256
    　
　參考:
    　 allocate(3), allocate_mapping(3), strlen(3)
　翻譯:
    　 Spock @ FF 97.Jul.25.　 (printed 3/16/95)
