to_int(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 to_int - 將一個浮點數或緩衝區 (buffer) 轉換為一個整數.
　語法:
    　 int to_int( float | buffer x );

    　 整數 to_int( 浮點數 或 緩衝區 x );

　用法:
    　 如果 x　 是一個浮點數, 則 to_int() 返回 x　 小數點之前的整數值. 如果 x　 是一個緩衝區, 則 to_int() 返回緩衝區之中的整數 ( 以 network-byte-order 的形式 ).
　參考:
    　 to_float(3), read_buffer(3), sprintf(3), sscanf(3)
