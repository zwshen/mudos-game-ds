unique_array(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 unique_array() - 將一個物件數組分組.
　語法:
    　 mixed unique_array( object *obarr, string separator, void | mixed skip );
    　 mixed unique_array( mixed *arr, function f, void | mixed skip );

    　 混合 unique_array( 物件 *obarr, 字符串 separator, 無參數 或 混合 skip );
    　 混合 unique_array( 混合 *arr, 函數指針 f, 無參數 或 混合 skip );

　用法:
    　 unique_array()　 將函數 separator　 返回同值的物件分為同組. obarr 只能是一個物件數組, 其他數據類型則予以忽視. 在 obarr　 中的每一個物件各會調用函數 separator 一次. 如果指定參數 skip,　 會先過濾掉符合 skip 的物件元素.
    　 而第二種語法的作用略有不同. 數組中的每一個元素將依次作為 f　 的參數, 元素仍依照 f　 的返回值分類. 但是數組不限定為物件類型.
　返回值:
    　 返回的值是含有多個數組的數組, 格式如下:

    　 ({
    　　 ({第一組:1, 第一組:2, .... 第一組:N }),
    　　 ({第二組:1, 第二組:2, .... 第二組:N }),
    　　 ({第三組:1, 第三組:2, .... 第三組:N }),
    　　　 ....
    　　 ({第 N 組:1, 第 N 組:2, .... 第 M 組:N }),
    　 })
