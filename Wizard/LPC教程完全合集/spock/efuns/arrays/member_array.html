member_array(3)　 MudOS v21c2　 (5 Sep a994)

　名稱:
    　 member_array() -　 在一個數組 (array)　 或字符串 (string) 中找尋指定的項目 (item) , 並返回此項目為第幾項的索引.
　語法:
    　 int member_array( mixed item, mixed * | string arr, void | int start );

    　 整數 member_array( 混合 item, 混合 * 或 字符串 arr, 無參數 或 整數 start );
     

　用法:
    　 返回 item 在數組 arr　 中第一次出現時, 出現在第幾項. 如果使用 start 參數, 則從 start 指定的第幾項之後找起. 如果找不到 item, 則返回 -1.
