sort_array(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 sort_array() - 將一個數組的內容排序.
　語法:
    　 mixed *sort_array( mixed *arr, string fun, object ob );
    　 mixed *sort_array( mixed *arr, function f );
    　 mixed *sort_array( mixed *arr, int direction );

    　 混合 *sort_array( 混合 *arr, 字符串 fun, 物件 ob );
    　 混合 *sort_array( 混合 *arr, 函數指針 f );
    　 混合 *sort_array( 混合 *arr, 整數 direction );

　用法:
    　 第一種語法會返回一個與 arr　 有相同元素的數組, 但是所有的元素順序依照 ob->fun() 所指定的規則依上升順序 (ascending) 排序. 每次調用 ob->fun() 時, 會傳入兩個參數. fun()　 應依照兩個參數間的關係返回 -1, 0, 1 (較小, 等於, 較大).

    　 第二種語法與第一種一樣, 只是使用函數指針 f　 代替字符串 fun.

    　 第三種語法使用內建的排序行程 (built-in sort routine). direction 為 1　 或是 0 ,　 將會依照上升順序快速排序, 而 direction 為 -1 將依照下降順序 (descending) 排序. 內建的排序行程有其限制, 即所有數組中的元素類型必須一樣 (homogeneous).　 而類型只能是字符串、整數、浮點數其中一種. 數組的數組將依照每個數組的第一個元素排序, 讓您可以對數據庫 (database) 排序.
