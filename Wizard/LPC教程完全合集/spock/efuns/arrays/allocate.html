allocate(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 allocate() - 配置內存給一個數組 (array).
　語法:
    　 mixed *allocate( int size );

    　 混合 *allocate( 整數 size );

　用法:
    　 配置內存給一個有 size 個元素 (element)　 的數組. 元素的數目必須大於 0, 而且不可以超過系統設定的上限 (通常約 10000 左右) . 所有元素的初始值 (initialized) 均為 0.
　參考:
    　 sizeof(3), allocate_mapping(3)
　翻譯:
    　 Spock @ FF 96.Oct.12.　 (printed 3/16/95)
