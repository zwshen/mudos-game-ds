children(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 children() - 返回一個指定物件 (object) 所復制出來的物件數組.
　語法:
    　 object * children( string name );

    　 物件 *children( 字符串 name );

　用法:
    　 這個函數返回一個物件數組 (array), 物件群則是文件名稱為 name 的物件所復制出來的. 一個使用的範例是找出所有從使用者物件 (user object)　 復制出來的物件:
    　
      　　 object *list;

      　　 list = children("/obj/user");

    　 這樣可以讓您找出所有的使用者 (包括已經斷線的使用者),　 而 users() 只會回報還能互動的 (interactive) 使用者.
　參考:
    　 deep_inherit_list(3), inherit_list(3), objects(3)
