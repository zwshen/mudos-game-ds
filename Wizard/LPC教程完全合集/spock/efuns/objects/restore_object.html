restore_object(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 restore_object() - 從一個文件讀回一個物件的變量值.
　語法:
    　 int restore_object( string name, int flag );

    　 整數 restore_object( 字符串 name, 整數 flag );

　用法:
    　 從叫作 name 的文件讀回目前物件的變量值. 如果 flag 是 1, 則非靜態變量 (non-static) 不會在讀回之前被清除為零. (普通狀況下, 非靜態變量都是零).

    　 如果讀回的過程出錯, 則所有的變量都不會更動, 並返回錯誤.
