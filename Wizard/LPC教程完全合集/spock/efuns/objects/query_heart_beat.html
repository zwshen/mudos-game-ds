query_heart_beat(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 query_heart_beat() - 查詢一個物件的心跳 (heart beat) 狀況.
　語法:
    　 int query_heart_beat( object );

    　 整數 query_heart_beat( 物件 );

　用法:
    　 對於已經調用過 set_heart_beat() 函數的物件, query_heart_beat() 返回其心跳值. 如果不指定 ob, 則預設為目前的物件. 沒有心跳的物件則返回 0.
　參考:
    　 heart_beat(4), set_heart_beat(3)
　翻譯:
    　 Spock @ FF 97.Apr.30.　 (printed 3/16/95)
