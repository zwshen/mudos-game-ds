reload_object(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 reload_object - 把一個物件還原成剛剛載入的狀態.
　語法:
    　 void reload_object( object ob );

    　 無返回值 reload_object( 物件 ob );

　用法:
    　 對 ob 調用 reload_object() 時, 所有由驅動程序管理的物件特性 (心跳 heart_beat,　 外部調用 call_outs,　 光線 light, 陰影 shadow 等) 和內部變量都會重新初始化 (re-initialized), 然後調用　 create(). 這個函數的效果相當於先摧毀再載入物件, 用不著存取磁碟.
　參考:
    　 export_uid(3), new(3), clone_object(3), destruct(3)
　翻譯:
    　 Spock @ FF 97.May.26.　 (printed 3/16/95)
