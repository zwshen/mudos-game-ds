move_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 move_object() - 把目前的物件搬到其他的環境 (environment) 中.
　語法:
    　 void move_object( object item, mixed dest );

    　 無返回值 move_object( 物件 item, 混合 dest );

　用法:
    　 把物件 item 搬到物件 dest 裡面. item 必須是 this_object(). 如果 item 是 this_object(), move_object() 可以只指定 dest 一個參數.
　參考:
    　 this_object(3), move(4)
