set_hide(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 set_hide() - 對一個可以藏匿的物件設定藏匿標誌 (hide flag).
　語法:
    　 void set_hide( int flag );

    　 無返回值 set_hide( 整數 flag );

　用法:
    　 以整數 flag 設定一個物件的藏匿標誌. 標誌只能是 0　 或 1, 分別代表取消和開始藏匿狀態. 只有在 master()->valid_hide(ob) 對此物體返回真值時, 此物體才能藏匿. 當物體處於藏匿狀態時, 只有其他有能力藏匿的物體才能使用 find_object()　 或返回多個物件的函數如 users(), children(), all_inventory() 等等找到此物體.
　參考:
