clonep(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 clonep() - 檢查給定的一個變量是否為復制的物件.
　語法:
    　 int clonep( void | mixed arg );
    　

    　 整數 clonep( 無參數 | 混合 arg );

　用法:
    　 如果參數 arg 有設定 objectp() 和 O_CLONE 標誌 (flag), 此函數就返回「真」 (true) 的值 (也就是 1). MUD 的驅動程序 (driver) 會對經由 new(3) (clone_object(3)) 創造出來的物件, 設定 O_CLONE　 標誌. clonep() 函數對於主物件 (master copy,　 經由 call_other(3) 載入) 不會返回「真」值. 請注意, 如果 clonep() 返回「真」值, 則 file_name()　 對此物件返回的文件名稱會加上 #n ( n 為整數). clonep() 不加上參數則預定物件為 this_object().
　參考:
    　 virtualp(3), userp(3), wizardp(3), objectp(3), new(3), clone_object(3), call_other(3), file_name(3)
