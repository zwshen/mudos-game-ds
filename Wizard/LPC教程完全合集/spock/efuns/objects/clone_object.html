clone_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 clone_object() - 載入一個物件的復制物件
　語法:
    　 object clone_object( string name );

    　 物件 clone_object( 字符串 name );

　用法:
    　 根據定義的 name 載入一個新的物件 (object) , 並且給此物件一個獨有的名稱. 此函數返回一個新的物件. 一個物件的 environment()　 返回值如果不為 0　 的話, 就無法復制.
　參考:
    　 destruct(3), move_object(3), new(3)
　翻譯:
    　 Spock @ FF 97.Jan.27.　 (printed 3/16/95)
