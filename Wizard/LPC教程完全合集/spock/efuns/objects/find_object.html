find_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 find_object() - 以指定的文件名稱尋找一個物件.
　語法:
    　 object find_object( string str );

    　 物件 find_object( 字符串 str );

　用法:
    　 用文件名稱 str　 尋找一個物件. 如果要尋找的物件是一個復制出來的物件, 亦可以用 file_name()　 返回的文件名稱作為參數來尋找.
　參考:
    　 file_name(3), stat(3)
