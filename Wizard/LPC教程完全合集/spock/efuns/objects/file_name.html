file_name(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 file_name() - 取得一個物件的文件名稱.
　語法:
    　 string file_name( object ob );

    　 字符串 file_name( 物件 ob );

　用法:
    　 file_name() 返回載入 ob 時所用的文件名稱. 如果物件是復制出來的, 則 file_name()　 返回的名稱並不是實際上磁碟中所用的名稱, 而是在原來的文件名稱之後加上「#」 號, 再加上物件的復制編號. 當遊戲啟動時, 物件的復制編號從 0　 開始起算, 每復制一個相同的物件就增加一號. 所以每一個復制出來的物件都有自己獨特的編號. 如果沒有指定 ob, 則 ob 預設為 this_object().
