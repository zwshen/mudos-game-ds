deep_inventory(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 deep_inventory() - 返回一個物件的內容物品, 如果物品中會以還含有物品, 將會以巢狀的方式繼續找下去.
　語法:
    　 object *deep_inventory( object ob );

    　 物件 *deep_inventory( 物件 ob );

　用法:
    　 返回 ob 含有之所有物品的數組 (array), 深層的物品也會繼續尋找下去.
　參考:
    　 first_inventory(3), next_inventory(3), all_inventory(3)
　翻譯:
    　 Spock @ FF 97.Feb.04.　 (printed 3/16/95)
