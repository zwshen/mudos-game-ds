save_object(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 save_object() - 將一個物件的變量值儲存到一個文件中.
　語法:
    　 int save_object( string name, int flag );

    　 整數 save_object( 字符串 name, 整數 flag );

　用法:
    　 把這個物件所有的非靜態變量 (non-static variables) 儲存到名叫 name 的文件中. 這個儲存的動作由主控物件 (master object) 中的 valid_write() 函數檢查是否允許這個物件寫入. 如果第二個參數指定為 1, 則變量值為零的的變量也會儲存進去 (一般說來不會是零). 而物件類型的變量則會儲存為 0.
　返回值:
    　 save_object() 成功會返回 1, 失敗就返回 0.
　參考:
    　 restore_object(3)
