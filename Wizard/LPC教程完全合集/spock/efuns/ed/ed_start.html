ed_start(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 ed_start() - 開始執行一個 ed 工作.
　語法:
    　 string ed_start( string file | void, int restricted | void )

    　 字符串 ed_start( 字符串 file 或 無參數, 整數 restricted 或 無參數 )

　用法:
    　 這個函數只有 __OLD_ED__ 不予定義 (define) 時有效.

    　 這個函數開始執行內部的文書編輯程序, 可以指定或不指定要予以編輯的文件 file.　 函數會返回執行完 ed 的編輯結果. 當 ed 程序仍在執行時, 可以使用 ed_cmd() 執行 ed 的命令.

    　 如果 restricted 的值為 1　 , 所有更改文件的命令都會被視為無效.

    　 一個物件同一時間內只能執行一個 ed 工作.
