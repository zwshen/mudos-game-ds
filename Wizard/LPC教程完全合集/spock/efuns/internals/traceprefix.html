traceprefix(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 traceprefix() - 決定該追蹤哪一個物件.
　語法:
    　 string traceprefix( string prefix );

    　 字符串 traceprefix( 字符串 prefix );

　用法:
    　 如果設定追蹤前置 (traceprefix) (不能為 0), 只有在遇到追蹤前置所指名的物件才進行追蹤.
　參考:
    　 trace(3)
　翻譯:
    　 Spock @ FF 97.Jul.25.　 (printed 3/16/95)
