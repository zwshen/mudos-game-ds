memory_info(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 memory_info - 取得一個物件或整個 mud 的內存使用量.
　語法:
    　 varargs int memory_info( object ob );

    　 不定參數 整數 memory_info( 物件 ob );

　用法:
    　 如果指定 ob,　 則 memory_info()　 返回 ob 正在使用的內存量. 如果沒有指定參數, 則返回整個 mud　 的內存使用量. 請注意, 整個 mud　 的內存使用量不是實際上系統配置給 mud 的內存量.
　參考:
    　 debug_info(3), malloc_status(3), mud_status(3)
