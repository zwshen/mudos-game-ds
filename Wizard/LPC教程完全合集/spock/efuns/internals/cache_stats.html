cache_stats(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 cache_stats() - 報告驅動程序 (driver) 和 mud 函數庫 (mudlib) 的變化統計數據.
　語法:
    　 void cache_stats( void );

    　 無返回值 cache_stats( void );

　用法:
    　 這個函數只有在編譯驅動程序時, options.h 中有定義 CACHE_STATS 才有用. 這個外部函數 (emulated function, 即 efun)　 會列出 call_other() 的快取 (cache) 效率.
　參考:
    　 opcprof(3), mud_status(3)
　翻譯:
    　 Spock @ FF 96.Oct.16.　 (printed 3/16/95)
