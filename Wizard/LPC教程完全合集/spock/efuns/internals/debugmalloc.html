debugmalloc(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 debugmalloc() - 將已配置的內存 (malloc'd memory) 信息傾印到一個文件裡.
　語法:
    　 void debugmalloc( string filename, int mask );

    　 無返回值 debugmalloc( 字符串 filename, 整數 mask );

　用法:
    　 這個外部函數 (efun) 必須在編譯驅動程序 (driver) 時, 於 options.h　 同時定義 DEBUGMALLOC 和 DEBUGMALLOC_EXTENSIONS　 兩者才有效. debugmalloc() 會傾印 DMALLOC()　 配置的內存區塊的信息及相關的 macros. (如果 macro 有 mask bitwise and'd (&) with the tag, 即 mask & tag　 不是零) 詳細情形請參閱驅動程序原始碼中的 md.c 及 config.h
　參考:
    　 set_malloc_mask(3)
　翻譯:
    　 Spock @ FF 97.Jul.23.　 (printed 3/16/95)
