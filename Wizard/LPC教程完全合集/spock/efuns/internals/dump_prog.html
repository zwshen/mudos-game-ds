dump_prog(3)　 MudOS v21c2　　 (5 Sep 1994)

　名稱：
    　 dump_prog() - 傾印/反組譯一個 LPC 物件
    　
　 語法：
    　 void dump_prog( object ob, int flags default: 0, string file );

    　 無返回值 dump_prog( 物件 ob, 整數 flags 預設值: 0, 字符串 file );
    　

　 用法：
    　 dump_prog() 傾印關於程序 obj 的信息到某個文件 `file'，若沒有指定 `file' 的話，則內定是 "/PROG_DUMP"。若調用這個函數的物件沒有這個文件之寫入權限的話，它將會失敗。

    　 flags 可以是下列數值的組合：

        1 - 包含 i-code 的反組譯；
        2 - 包含行號信息。
    　
　 參考：
    　 debug_info(3), dumpallobj(3)
