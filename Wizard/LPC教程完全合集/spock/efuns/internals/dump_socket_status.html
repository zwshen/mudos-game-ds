dump_socket_status(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 dump_socket_status() - 列出每一個 LPC 的 socket 狀態
　語法:
    　 void dump_socket_status( void );

    　 無返回值 dump_socket_socket( 無參數 );

　用法:
    　 dump_socket_status()　 是一個診斷的功能, 列出 MudOS　 驅動程序組態之所有 LPC socket 的目前狀態. 此函數常用於 LPC socket　 應用程序 (application)　 的除錯. 顯示的每一行各對照一個 LPC socket.　 第一列是 LPC socket 的第一個描述值 (descriptor), 第二列是第二個描述值, 以此類推. socket　 的總數在編譯驅動程序時已經固定.

    　 第一列 Fd 是與 LPC socket 相關的作業系統的文件描述值. State 是目前的 LPC socket 運作情形. Mode　 是 socket 模式 (mode) , 是 socket_create()　 的參數之一. Local 和 Remote address　 是 Internet 位址和連接埠 (port number), 以 Internet 的分點表示法 (dot notation, 即 aaa.bbb.ccc.ddd 的格式) 顯示. * 表示一個位址或是該處的值為零. 請注意, CLOSED　 狀態的 LPC socket 為尚未使用的 socket, 所以該 socket　 顯示的數據可能會特異.

    　 以下的輸出數據由 Portals mud　 產生, 當時唯一的 socket 應用程序 (application)　 為 MWHOD. 數據顯示目前正在使用兩個 socket.　 一個是 STREAM 模式的 socket 正在傾聽連線要求. 另一個是 DATAGRAM 模式的 socket 正在等待數據輸入.
     
    Fd State Mode Local Address Remote Address
    13 LISTEN STREAM *.6889 *.*
    14 BOUND DATAGRAM *.6888 *.*
    -1 CLOSED MUD *.* *.*
    -1 CLOSED MUD *.* *.*
