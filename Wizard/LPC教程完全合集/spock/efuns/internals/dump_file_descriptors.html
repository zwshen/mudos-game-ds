dump_file_descriptors(3) MudOS v21c2　 (5 Sep 1994)

　名稱：
    　 dump_file_descriptors() - 傾印 MudOS 程序的文件描述值表
　語法：
    　 void dump_file_descriptors( void );

    　 無返回值 dump_file_descriptors( 無參數 );

　用法：
    　 提供這個函數是為了幫助 MudOS 驅動程序的除錯，和克服在一些未提供等效或是出眾除錯機制之 UN*X 作業系統在執行上的不足之處。這個函數的輸出結果是和系統十分相關的。每個文件描述值都會被檢查是否參考到一個已開啟的文件。如果是的話，將會顯示由系統調用 fstat() 得到的 stat 結構信息。
