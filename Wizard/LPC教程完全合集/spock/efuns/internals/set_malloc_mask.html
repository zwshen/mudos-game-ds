set_malloc_mask(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 set_malloc_mask() - 設定控制內存配置的除錯數據顯示的遮罩 (the mask controlling display of malloc debug info).
　語法:
    　 void set_malloc_mask( int mask );

    　 無返回值 set_malloc_mask( 整數 mask );

　用法:
    　 這個外部函數 (efun) 只有在編譯 mud　 驅動程序 (driver) 時, options.h 中 DEBUGMALLOC　 和 DEBUGMALLOC_EXTENSIONS 皆同時定義時才有效. 此遮罩 (mask) 控制驅動程序配置內存和釋放內存時, 顯示多少與內存相關的除錯消息. 詳細數據請參閱驅動程序原始碼中的 md.c
　參考:
    　 debugmalloc(3)
　翻譯:
