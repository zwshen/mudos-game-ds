rusage(3)　 MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 rusage() - 報告以系統調用 (system call) getrusage() 收集的信息.
　語法:
    　 mapping rusage( void );

    　 映射 rusage( 無參數 );

　用法:
    　 這個外部函數 (efun) 通過 getrusage() 的系統調用收集信息. 請參閱 getrusage() 線上手冊的說明以查詢收集的信息種類. 有的系統沒有 getrusage(), 但是有 times() 的系統調用. 在這種系統裡只有 utime 和 stime 兩項數據. 報告的時間單位是微秒 (millisecond).

    　 以下是 rusage() 的範例:
    　

      　 void
      　 create()
      　 {
        　　 mapping info;

        　　 info = rusage();
        　　 write("使用者時間 = " + info["utime"] + "微秒\n");
        　　 write("系統時間 = " + info["stime"] + "微秒\n");

      　 }
    　 可以使用的項目有:
    　
      　 utime, stime, maxrss, ixrss, idrss, isrss, minflt, majflt, nswap, inblock, oublock, msgsnd, msgrcv, nsignals, nvcsw, nivcsw.
　參考:
    　 time_expression(3), function_profile(3), time(3), uptime(3)
　翻譯:
