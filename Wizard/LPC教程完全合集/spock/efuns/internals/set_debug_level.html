set_debug_level(3) MudOS v21c2　 (5 Sep 1994)

　名稱:
    　 set_debug_level() - 設定驅動程序 (driver) 使用 debug() 宏 (macro) 時, 其除錯 (debug) 的等級.
　語法:
    　 void set_debug_level( int level );

    　 無返回值 set_debug_level( 整數 level );

　用法:
    　 這個外部函數 (emulated function, efun) 只有在編譯驅動程序時使用 -DDEBUG_MACRO 才有效. set_debug_macro() 讓 mud　 執行時, 可以從內部控制除錯消息的量及種類.

    　 請參考驅動程序原始碼中的 debug.h, 其中有詳細的定義.
