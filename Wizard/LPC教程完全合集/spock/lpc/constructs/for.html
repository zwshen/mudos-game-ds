* LPC 的 for 循環 :

LPC 的 for 循環與 C 提供的相同. 語法如下:

    for ( 運算式 0; 運算式 1; 運算式 2 ) {
      語句;
      ...;
    }
執行循環之前, 會先計算「運算式 0」. 每次循環周期開始時, 計算「運算式 1」. 如果「運算式 1」計算的結果得到 0, 則循環停止. 「計算式 2」則於每次循環周期結束時計算之.

循環裡面的 break 會讓循環停止. continue 則會跳過此次循環周期剩餘的部分, 從下一個周期開始 (會先計算「運算式 2」).

典型的 for 循環, 用以固定的次數重覆執行一段循環中的語句.

範例:

    int i;
    for (i = 0; i < 10; i++) {
      write("i == " + i + "\n");
      write("10 - i == " + (10 - i) + "\n");
    }
