* if 與 else 語句:

LPC 的 if 語句與 C 提供的相同. 語法如下:

    if (運算式)
      語句;
第二種語法:
    if (運算式) {
      語句;
    }
第三種:
    if (運算式一) {
      　語句一;
    } else {
      　語句二;
    }
第四種:
    if (運算式一) {
      語句一;
    } else if (運算式二) {
      運算式二;
    }
else if 子句沒有什麼數量上的限制.

另一個好用的程序結構是 ? : 運算符. 它與 C 提供的一樣. 語法如下:

　　　 運算式一 ? 運算式一為真則進行運算式二 : 運算式一為假則進行運算式三

某些情形下, ? : 算是以下程序結構的縮寫:

　　　 if (運算式一)
　　　　　　　 變量 = 運算式二;
　　　 else
　　　　　　　 變量 = 運算式三;

上面這段程序可以轉換成下面這行:

　　　 變量 = 運算式一 ? 運算式二 : 運算式三;
