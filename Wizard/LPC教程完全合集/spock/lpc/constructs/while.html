LPC 的 while 循環:

LPC 的 while　 循環與 C　 提供的相同. 語法如下:

    while (運算式)
      語句;
語句可以換成一段用 {} 大括號圍起來的程序. 舉例如下:
    while (運算式) {
      語句一;
      語句二;
    }
只要運算式得到的結果為非零值 (non-zero),　 就會重覆執行 while　 循環裡面的語句. 如果在執行一次循環周期之前, 運算式得到零, 就不會執行裡面的語句. 在循環中如果有 break; 語句, 則此 while　 循環終止 (跳過所有尚未執行的語句). 循環裡面如果有 continue;　 語句, 則會從頭開始執行循環 (跳過此周期中還未執行的語句).
　
    int test(int limit)
    {
      total = 0;
      j = 0;
      while (j < limit) {
        if ((j % 2) != 0)
          　continue;
        total += j;
        j++;
      }
      return total;
    }
此段源代碼的結果, 是把 0　 到 limit - 1　 之間的偶數全部加起來.
