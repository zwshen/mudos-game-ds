* switch 語句

LPC 的 switch 語句與 C　 的 switch 語句幾乎一樣. 唯一不同的地方是 LPC　 的 switch　 判斷依據的運算式可以使用字符串或整數.
語法如下:

    switch (運算式) {
      case 常數一 : 語句一;
        break;
      case 常數二 :　 語句二;
        break;
      default :　 語句三;
        break;
    }
switch 可以代替一堆 if else if else if else 結構. 上面的 switch 等於:
    tmp = 運算式;

    if (tmp == 常數一) {

      語句一;
      ...;
    } else if (tmp == 常數二) {
      語句二;
      ...;
    } else {
      語句三;
      ...;
    }
switch　 與 if 語句之間最大的不同處, 在於某個 case 最後如果沒有 break; 語句, 則會執行下一個 case.
