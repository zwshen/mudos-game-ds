buffer

緩衝區 (buffer) 介於 LPC　 數組 (array) 形態和 LPC 字符串形態之間.
緩衝區的目的是便於操作二進位數據.
緩衝區不是「零值終止」(zero-terminated) む就是說, buffer　 有一個聯合長度 (associated length)め.
緩衝區是一個字節 (bytes)　 的數組, 而每個元素只有一個字節.

    buf[i] = x 和 x = buf[i];
    sizeof(buf);
    bufferp(buf);
    buf[i..j];
    buff = read_buffer(file_name, ...); (參數與 read_bytes 相同)
    int write_buffer(string file, int start, mixed source);
    buf = buf1 + buf2;
    buf += buf1;
    buf = allocate_buffer(size);
以上這些語法都能使用.

socket 的外部函數已經過修改, 可以接受並返回緩衝區形態. (STREAM_BINARY (3) 和 DATAGRAM_BINARY (4)　 模式.)
