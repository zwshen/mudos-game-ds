    目前大致上已將 MudOS v21c2 之原文文件翻譯完成。不過，因為譯者所學不多, 難免有遺珠之憾。未能翻譯之文件清單列於附註一。

    這份文件中的分類結構如下:
    　

    • applies –– 驅動程序對各個物件調用的函數
        • general –– 對一般物件調用的函數
        • interactive –– 對玩家物件調用的函數
        • master –– 對主控物件調用的函數
    • concepts –– 撰寫 LPC 所必備的一些概念
    • lpc –– 有關於 LPC 語法的資料
        • constructs –– LPC 程序的保留字
        • preprocessor –– LPC 程序的預處理命令
        • types –– LPC 的變量、函數的類型定義
    • efuns –– 各個外部函數的使用手冊
        • arrays –– 數組
        • buffers –– 緩衝區
        • calls –– 調用
        • compile –– 編譯
        • ed –– ed 編輯程序
        • filesystem –– 文件系統
        • floats –– 浮點數
        • functions –– 函數
        • general –– 一般
        • interactive –– 互動物件
        • internals –– 內部
        • mappings –– 映射
        • mudlib –– 函數庫
        • numbers –– 數字
        • objects –– 物件
        • parsing –– 語法分析
        • sockets –– 套接字
        • string –– 字符串
        • system –– 系統
    文件中如有錯誤或是更好的翻譯，請來函指正。如果您願意翻譯附註一中所列之文件，也請來函告知，並允許譯者收錄於往後的版本中。
