ARCH，ADMIN命令集
 2002-5-29  


1、callouts命令
指令格式：callouts [<函數名稱>]
缺省將列出系統中所有的延遲呼叫函數
否則列出指定的延遲呼叫函數.
2、chinese命令
指令格式：chinese <英文>==<中文>
設定中英文等價關係

3、cleanup命令
指令格式：cleanup <檔名>
對所有指定檔名的物件呼叫 clean_up，　且清除所有被復制出來的，但是又沒有
被放在另一個物件中的散失物件
　
4、force命令
指令格式：force <某人> to <指令>
強迫某人做某事, 但你的巫師等級必須比對方高, 而且必須是在同一個房間裡?
5、overview命令
指令格式：overview <主題>
縱觀整個mud中的情況
主題有:
user 共有幾個使用者物件
living 共有幾個生物(含使用者)物件
char 共有幾種人物物件 master copy 被載入
npc 共有幾個 NPC 物件
combat 共有幾個人物正在戰鬥狀態
kill 共有幾個人物正在性命相搏的戰鬥狀態
hunt 共有幾個人物正和其他人物互相仇恨
heart beat 共有幾個物件的 heart beat 為有效狀態
virtual 共有幾個虛擬物件
room 共有幾個房間
special room 共有幾個物件繼承標準房間
equip 共有幾個物件繼承標準裝備
object 共有幾個物件被載入　

6、possess命令
指令格式：posssess <目標生物>
附身到另一個生物上，這個生物必須和你有相同的 EUID　
附身後用 quit 指令會回到原來的身體　

7、promote命令
指令格式 : promote <某人> (權限等級)
用來提升權限等級, (player) (immortal) (apprentice) (wizard)
(arch) (admin)
8、purge命令
指令格式：purge [<未上線天數>] | <使用者姓名> because <原因>
清除一個使用者或清除超過一定天數不曾上線地使用者　

9、reboot命令
指令格式: reboot
重新起動遊戲, 系統會開始倒數計時, 十五分鐘後重新起動　

10、reclaim命令
指令格式 : reclaim
用處 : 將存在於記憶體裡無用的變數清除掉, 以減少記憶體的使用　

11、rehash命令
指令格式：rehash <目錄名稱>
這個指令會更新系統內部的指令表，如果你新增或刪除了某個目錄下的指令，必杏
? 這個指令更新該目錄才能使用　

12、wizlock命令
指令格式: wizlock <巫師等級>
限制某個等級以下的使用者進入遊戲　
其中巫師等級分別為 (immortal) (apprentice) (wizard) (arch) (admin), 若胰
? 消限制, 則輸入 (player).


13、disassemble命令
指令格式：disassemble <物件> <輸出檔名>
反組譯一個物件並輸出至指定文件

14、dump命令
指令格式 : dump
將所有物件的狀態記錄至根目錄下的文件OBJ_DUMP中。

15、profile命令
指令格式: profile

16、shutdown命令
指令格式: shutdown
馬上重新起動遊戲
