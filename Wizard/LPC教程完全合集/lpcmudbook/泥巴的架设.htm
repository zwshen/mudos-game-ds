泥巴的架设
 2002-7-31  


1.首先 download Mudos 和 Mudlib:


在什么地方有单机的mudlib下载这点不需要说明了，现在提供MUDLIB资源的网站实在是多如牛毛，去门户站点搜索关键字就可以了，不过江南提醒您注意一下：一定要下比较完整，版本比较新的MUDLIB，这样的MUDLIB自身的BUG少，下载后的除错(debug)工作也相对轻松些，可以很快架设线路。





2.设置配置文件 config.sys (在每一个MUDLIB中都有相应的config.sys，这个文件与mudos.exe是同等重要的)


Win95/NT 下:


修改里面的:


name: 你自己的MUD名字 (如:爱你爱到杀死你)


port number : 4000 (端口号可改成自己喜欢的数字如: 7777,6666 etc.)


mudlib directory : d:/lpmud/lpmud (一定要和自己的机器上MUDLIB存放路径一致)


global include file : "/include/globals.h"


UNIX,Linux 下:


修改里面的:


name: 你自己的MUD名字 (如:爱你爱到杀死你)


port number : 4000 (端口号可改成自己喜欢的数字如: 7777,6666 etc.)


mudlib directory : /lpmud/mudlib (可根据自己机上的 mudlib 的路径修改)


binary directory : /lpmud/bin (这一行是 MUD driver 的执行档目录)


global include file : "/include/globals.h"


3.修改Mudlib中的 adm/etc/wizlist


加入你所要用的 id (等级) 例如 winding (admin)，这里要说明一下，应当首先进入MUD登陆一个ID后再用wizlist修改较好。


4. 把 /data/login,/data/user,/data/board,/data/mail里的内容都删掉


这是由于 MudOs 的不同,可能对数据的存储格式不全一样,删掉就不会有问题了.这一步一般来说不用做到，因为现在的MUDLIB基本上没有太大的这方面的差别，不过如果有的话那就一定要做这一步了。


5.运行MUD


运行mudos.exe就可以了，在你的windows的右下角任务栏里是不是多了一个地球的图标呢，呵呵。


6.telnet MUD


Ip : localhost or 127.0.0.1 或是自己实际的 IP Port: 4000


用telnet 软件 zmud,netterm 都行.


用你自己设好的 id 登录,理论上就可以用了


以天神(admin)的 id 提升其他角色的方法: promote <id> (等级)


例如: promote tying (admin)
